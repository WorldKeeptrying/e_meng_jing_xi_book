# 苏安-人设

## 基本信息
- 姓名：**苏安**
- 性别：男
- 昵称：大兄弟（姐姐称呼）
- 姐姐：[[苏小小]]
- 关系：苏小小的双胞胎弟弟

## 背景故事
- 与姐姐苏小小很小就相依为命
- 头部发育不完全，外人看来有些痴傻
- 唯一信任的人是姐姐苏小小
- 在苏安眼中，世上只有两种人：姐姐，和除姐姐外的其他人

## 特殊能力：共情

### 能力描述
- **万物共情**：可以与任何有生命的东西共情，深入它们的感受
- 能理解对方的处境、心境，甚至"心声"
- 无论是猛虎还是一株野草，都能产生共情

### 能力表现
1. **海洋公园事件**：看海豚表演时泪流满面，因为感知到"那些大鱼好可怜，它们还饿着肚子"
2. **踩倒的花**：扶起被踩倒的花，严肃告诉姐姐"它被你吓到了，还弄疼了"
3. **虎园事件**（最典型）：
   - 7-8岁时坠入老虎虎园
   - 被数只老虎围住但没有发动进攻
   - 虎群领袖凑近后，苏安像对待家猫一样抓它耳朵、摸它头
   - 老虎将大脑袋挤进苏安怀里，发出类似哭泣的抽泣声
   - 苏安说："那只大猫说这里不是它的家……它想回家"

### 对人的判断：色彩感知

苏安用颜色来区分人的"本质"：

| 人物类型 | 颜色 | 示例 |
|---------|------|------|
| 刚出生的婴儿 | **纯白色** | 最初级、最单纯 |
| 普通佣人/管家 | 白色偏灰 | - |
| 有心机的人 | 白中带灰 | 管家级别 |
| 机车女 | 白中偏灰 | - |
| 梁龙 | 白中偏红 | - |
| 杀人犯 | 淡淡的红色 | 手上有人命 |
| **尘然** | **深红色** | 极为罕见，来自"深红"组织 |
| **江城/郝帅** | **黑色** | 从未出现过，古怪至极 |

### 能力局限性
- 表达能力有限，姐姐需要猜测他的意思
- 会给感兴趣的事物编号（复杂古怪的数字）
- 曾请密码学家破译，失败告终
- 对老女人的判断出现异常：白色带灰 → 纯白 → 无色（比婴儿还"空白"）

## 关于"鲸歌"
- 能听到姐姐苏小小听不到的**鲸歌**
- 鲸歌声空旷、寒寂、幽然
- 感知到"被沉入北冰洋水下，透过厚冰层仰望外面世界"的感觉
- 每次听到鲸歌都伴随着某种危险的预兆

## 关于森林的认知
- **森林是活的**
- 目睹一拨又一拨人进入森林
- 船员、黑石镇居民……都是祭品
- 还有无法描述、无法预见、无法抗衡的**"它"**

## 当前状态
- 留在灯塔内养伤
- 姐姐苏小小离开前说"去去就回"
- 独自面对老女人的威胁

## 新增信息（第二百一十至二百一十一章）

### 遭遇老女人/影子攻击
- 听到"咔咔"声，像是皱巴巴的东西展开身体，又像生物褪壳
- 老女人躲在阴暗角落，布满血丝的眼睛盯着他
- **影子开始独立行动**：
  - 从墙壁上延伸，越来越高（2米→3米→4米→超过石室高度）
  - 老女人本体不动，但影子向苏安靠近
  - 影子是透明/无色的存在
- 苏安感知到：墙上留下影子的家伙是**肉眼不可见的存在**
- 苏安**不恐惧**，如同当年面对虎群一样
- 站起身伸出手，像在等待"大猫咪"把头探进怀里

### 感知到危险
- 鲸歌再次回响
- 感知到老女人与影子的双重存在
- 展现了不惧危险的一面
