# 第二十三章 药 - 剧情总结

## 核心事件
- 饭后胖子发现江城囤积大量**外伤药和镇痛药**
- 江城认真思考后拿出镇痛贴审视胖子
- 江城外出购买更多药品
- 胖子怀疑江城要**囚禁虐待**自己
- 江城看穿胖子心思，直言没人会对囚禁胖子感兴趣
- 江城解释药品用途：**捆、封口、镇痛、亢奋、致幻**
- 胖子三观第三次碎裂

## 关键信息

### 江城购买的药品
| 药品 | 用途 |
|------|------|
| 绷带 | **捆住胖子** |
| 医用胶带 | **封嘴防止叫喊** |
| 镇痛剂 | 遭受虐待时保持清醒 |
| 亢奋剂 | 两盒，不好买，价格贵 |
| 致幻药 | 混在水里喝 |

### 胖子心理活动
| 阶段 | 想法 |
|------|------|
| 发现药品 | 江城身体不像需要这些 |
| 江城审视自己 | 不祥预感 |
| 江城出门 | 怀疑要囚禁自己 |
| 推理 | 江城和别墅一家是一类人 |
| 决定逃跑 | 找理由离开 |
| 江城放话 | "出了这扇门就别想回来" |
| 选择留下 | 彼此照应 |
| 听到药品用途 | 震惊（"？？？"） |

### 江城的解释
- **绷带**：用来捆住你
- **医用胶带**：封住你的嘴防止叫喊引来警察
- **镇痛剂**：保证你遭受虐待时保持清醒
- **亢奋剂**：不好买，价格贵，看造化
- **致幻药**：混在你喝的水里

### 江城的态度
- "没人会对囚禁虐待一个胖子感兴趣"
- "腿长在你身上，你随时可以走"
- "只要出了这扇门，你就不用想着回来了"
- 解释时眼神"像问你真想知道"
- "千万别将这个秘密说出去"

### 胖子的推测
- 江城可能患有**难以启齿的隐疾**
- 面对陈晓萌说过"30秒就足够了"
- 上天公平，每个人都有痛楚
- 眼神充满鼓励

### 江城性格
- **买药**：直接告诉胖子放柜子里
- **漫不经心**：敲键盘漫不经心
- **看穿人心**：知道胖子在想什么
- **威胁与放养并存**：说出门就别回来，但确实让走
- **认真思考**：听到药品问题真的思考
- **无奈**：叹了口气告诉胖子

## 悬念
1. 江城买药真正目的是什么？
2. "遭受虐待"是指什么？噩梦中的虐待？
3. 江城真的会囚禁胖子吗？
4. "出了这扇门就别想回来"是威胁还是保护？
5. 江城是否有隐疾？
