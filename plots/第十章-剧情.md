# 第十章 - 剧情总结

## 核心事件

### 真相还原

**江城躲藏过程**：
- 江城轻手轻脚起身，躲进卫生间
- 胖子看到的"睡"在床边地板上的身影是影子（鬼）
- 真正的江城在卫生间门后，趁胖子发呆将他拉进去

### 现场勘查

**水渍线索**：
- 床上一片狼藉
- 地板上留着一滩滩水渍
- 水渍从床延续到门，走廊里也有
- 水渍停在走廊尽头锁着的房门前

**谢雨死亡确认**：
- 已死亡，无人生还奢望
- 被鬼裹在被子里带走，拖向锁着的房间

### 推理确认

**江城推理**：
- 鬼是四口之家的那个女孩
- 证据：鬼穿着女人的休闲鞋，在一层鞋柜见过
- 陈晓萌推理：家庭成员虐待女孩致死，女孩回来复仇

**暖姐观点**：
- "你是要和鬼讲道理吗？"
- 暗示：鬼不会讲道理，所有人都有危险

### 锁着的房间探索

**团队决定**：
- 樊力提议查看锁着的房间
- 所有人一同前往（分开更危险）
- 梦界规则：鬼极少短时间内接连杀人

**门的状态**：
- 锁被打开了（之前一直上锁）
- 樊力轻轻一推就开了

**房间特征**：
- 面积不大（男孩卧室的三分之二）
- 无窗户，需要开灯
- 江城找到开关（靠近门位置墙上）

### 谢雨尸体发现

**尸体状态**：
- 蜷缩在被子里
- 眼睛突出，瞳孔放大，脸色发青
- 面部及颈部肌肉扭曲
- 已死亡

**尸体姿势**：
- 两只手臂僵直，呈爪状护在胸前（防御姿态）
- 手背上青筋绷起老高

**身上伤痕**：
- 布满血痕（鞭子一类抽打）
- 几处锐器伤口
- 两手被手铐紧紧拷在一起

### 虐待工具

**背包内容**（暖姐找到）：
- 皮鞭
- 匕首
- 麻绳（与库房找到的绳子一样）
- 手铐
- 几根粗细不同的蜡烛

**江城推理**：
- 伤痕规整，无挣扎扭曲痕迹
- 谢雨遭受虐待时已无意识
- 先被吓死，然后被拖来虐打（鞭尸）

### 鞋码之谜（关键线索）

**鞋子异常**：
- 谢雨脚上的运动鞋不是他之前穿的
- 运动鞋很普通，有些破旧，十分臃肿
- 与文员打扮不搭调

**双层鞋子**：
- 运动鞋里还包裹着一双皮鞋
- 皮鞋被挤压变形
- 谢雨脚背诡异扭曲（骨头揉碎后硬塞）
- 皮鞋才是谢雨自己的鞋

**运动鞋归属**：
- 运动鞋是四口之家中男孩的
- 但尺码与一层鞋柜找到的运动鞋相差很大
- 不像同一个人所有

**暖姐推测**：
- 别墅里有三个男人
- 一位男主人 + 两个儿子
- 修正之前的"四口之家"认知

## 角色表现

### 江城
- **冷静到极致**：面对尸体仍能分析伤痕
- **推理能力**：通过鞋子判断鬼的身份
- **观察力**：发现鞋子不对，指出鞋码差异
- **职业暴露**：夜场话题试图带偏，被樊力阻止
- **防护意识**：让暖姐用刀挑开鞋带，不直接接触

### 樊力
- **领导力**：组织搜索锁着的房间
- **压制恐惧**：将心中寒意压制住
- **谨慎**：用刀挑鞋带，避免直接接触
- **团队管理**：及时阻止话题跑偏

### 暖姐
- **冷静**：亲手掀开被子查看尸体
- **专业**：找到装着虐待工具的背包
- **推理**：提出三个男人的推测
- **脸色奇怪**：对推测结果感到不安

### 陈晓萌
- **恐惧**：捂嘴惊呼
- **推理辅助**：提出男孩成长过程中穿过的假设
- **法律知识**：协助分析案情

### 胖子
- **恐惧延续**：声音颤抖，哭丧着脸
- **吐槽**：怀疑江城是凶手
- **新人反应**：无法看清鬼的样子（太黑+太害怕）

## 新增线索与规则

### 虐待工具
| 工具 | 位置 | 用途 |
|------|------|------|
| 皮鞭 | 背包 | 抽打 |
| 匕首 | 背包 | 锐器伤口 |
| 麻绳 | 背包 | 捆绑（与库房绳子一样） |
| 手铐 | 背包 | 铐住双手 |
| 蜡烛 | 背包 | 用途不明 |

### 死亡规律
- 鬼极少短时间内接连杀人
- 睡着的人最容易被袭击
- 鬼会"鞭尸"（死后继续虐待）

### 家庭结构修正
- **原推测**：四口之家（父母+男孩+女孩）
- **新推测**：五口之家（父母+两个儿子+女孩）
- **证据**：两双不同尺码的男式运动鞋
- **疑问**：之前只找到四双鞋，第三双男鞋在哪？

### 门锁异常
- 锁着的房间门被打开了
- 谁打开的？（鬼？任务机制？）
- 房间可能是虐待室

## 悬念

1. 第三个男人（第二个儿子）在哪？
2. 为什么之前只找到四双鞋？
3. 虐待室的锁是怎么打开的？
4. 蜡烛的用途是什么？
5. 江城"夜场"经历是否暗示什么？
6. 鬼接下来会 targeting 谁？
