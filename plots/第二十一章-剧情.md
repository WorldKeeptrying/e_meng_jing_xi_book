# 第二十一章 皮阮 - 剧情总结

## 核心事件
- 皮阮咨询**女朋友的病情**（怀疑他动机不纯、有外遇）
- 胖子误会皮阮有"病"（精神病），闹出尴尬
- 江城给皮阮女朋友**介绍富婆对象**
- 皮阮女朋友"80大寿"邀请江城参加
- 皮阮出门后**左手臂环秘书腰肢**离开
- 江城展示**富婆客户资料**（赵桂琴89岁、刘牛马77岁）
- 江城承认给富婆"治病"（**夜里空虚寂寞冷**）
- 胖子三观碎裂

## 关键信息

### 皮阮的女朋友
| 属性 | 描述 |
|------|------|
| 症状 | 怀疑皮阮不是真心，在外面有别人 |
| 诊断 | 认为皮阮动机不纯，另有图谋 |
| 江城治疗 | **介绍富婆对象**（皮阮不知情） |
| 80大寿 | 后天举办，邀请江城参加 |

### 江城的治疗方法
江城表面是心理医生，实际上是**富婆中介/红娘**：
| 客户 | 年龄 | 资产 | 状态 |
|------|------|------|------|
| 赵桂琴 | 89岁 | 9栋别墅，豪车无算，资产过亿 | 单身 |
| 刘牛马 | 77岁 | 名下几十套公寓 | 单身，病重，无儿女 |
| 其他 | 未知 | 未知 | 夜里空虚寂寞冷 |

### 江城的治疗逻辑
- 富婆"夜里空虚寂寞冷" = 病症
- 皮阮类年轻男人 = "药方"
- 皮阮女朋友怀疑他 = 因为**她才是被介绍的那个**
- 江城："见到久违的笑脸荡漾在她们脸上，我也发自内心的快乐"

### 胖子三观碎裂过程
1. 误会皮阮有精神病 → 尴尬
2. 发现皮阮有秘书 → 三观碎裂
3. 发现江城做富婆中介 → 三观碾成粉末
4. 看到江城给富婆"看病"资料 → 震撼

### 皮阮的"病情"
| 表面症状 | 真实原因 |
|----------|----------|
| 女朋友怀疑他有外遇 | 女朋友被介绍了其他对象 |
| 付出多得不到理解 | 女朋友心思在富婆们身上 |
| 动机不纯 | 女朋友确实动机不纯（图富婆钱） |

### 胖子反应
- "不想努力了" → 江城介绍89岁富婆 → "还可以再努力一把"
- 问江城收了多少回扣 → 江城不理会上楼
- 对江城的工作充满好奇

### 江城性格
- **幽默/黑色幽默**：给富婆"治病"，介绍年轻男人
- **不解释**：被误会也不多做解释
- **神神秘秘**：上楼不知道忙什么
- **贪吃**：惦记排骨
- **原则**："我不是皮阮，我有我的原则"（但实际上在做同样的事）

## 悬念
1. 皮阮知道女朋友被介绍富婆吗？
2. 江城"神神秘秘"上楼做什么？
3. 胖子对江城的认知完全颠覆
4. 江城到底收了多少回扣？
