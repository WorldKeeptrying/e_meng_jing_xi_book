# 第六章-剧情

## 核心事件：探索线索与禁忌规则

### 死亡时间推断
- 壮汉通过怀表计时（7小时）推断死亡时间在凌晨2点-5点间
- 守夜顺序印证：江城→中年→胖子（被尿憋醒发现尸体）

### 禁忌规则（重要！）
- **核心规律**：这个世界不存在无缘无故被杀，死者一定触碰了某些**禁忌**
- **禁忌可能是破解任务的关键线索**
- 鬼极少在短时间内接连杀人

### 关于中年死亡的讨论
| 观点 | 提出者 | 内容 |
|------|--------|------|
| 去过一层卫生间是禁忌 | 清纯女 | 去过的都要死 |
| 反驳 | 胖子 | 他也去过一层卫生间（昨天搜索时） |
| 补充 | 壮汉 | 几人都去过一层卫生间，不该只死中年 |
| 解释 | 唇边痣女人 | 可能只是分先后，鬼不短时间内连杀 |

### 任务确认
- 车只是交通工具，**真正的任务是别墅**
- 大门被锁住，无法离开
- 需要破解别墅内曾发生的事情

### 搜索分组（老人带新人）
| 组别 | 人员 | 负责区域 |
|------|------|----------|
| 壮汉组 | 壮汉 + 胖子 | 客厅及门附近 |
| 女人组 | 唇边痣女人 + 男文员 | 偏远库房 |
| 江城组 | 江城 + 清纯女 | 厨房和餐厅 |

### 发现的线索
1. **麻绳**（女人组发现）：
   - 很粗的麻绳
   - 一端有**黑红色印记 = 血迹**
   - 位置：一摞木柴后

2. **枫叶胸针**（女人组发现）：
   - 淡绿色，枫叶形状
   - 主人应该是个**温婉的女孩**

3. **隐藏鞋柜**（壮汉组发现）：
   - 暗门按钮嵌在门边油画中（淡蓝色，与背景融为一体）
   - 三层鞋柜，一二层隔板已断
   - **鞋子清单**：
     - 2双男士皮鞋（男主人？）
     - 1双女士红色高跟鞋（女主人？）
     - 3双女款休闲鞋（其中2双属于较年轻女孩，尺码一致）
     - 1双男款运动鞋（尺码与皮鞋相差较多，可能是男孩）

### 别墅住户推断（四口之家）
- **男主人**：皮鞋
- **女主人**：红色高跟鞋
- **女孩**：2双休闲鞋（年轻）+ 枫叶胸针 → 温婉
- **男孩**：运动鞋 + 休闲鞋

### 二楼房间对应
- 主卧室 → 男女主人
- 小卧室1 → 女孩
- 小卧室2 → 男孩
- 上锁房间 → 未知
