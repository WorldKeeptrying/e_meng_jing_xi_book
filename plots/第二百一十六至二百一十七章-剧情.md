# 第二百一十六至二百一十七章 舒服吗/石画

## 章节概述

江城和胖子逃离石盘后发现没有追兵，在森林边缘遭遇尘然。江城与尘然表面称兄道弟，实则互相提防。双方来到石柱旁查看暗纹线索，发现9幅石画，前8幅描绘不同场景，第9幅为残图。胖子惊觉残图中的人影疑似老女人，推测她可能不是玩家而是特殊NPC，关联着黑石镇的秘密。

## 剧情详述

### 第二百一十六章 舒服吗

#### 逃离石盘
- 江城和胖子冲出森林，发现**没有东西追来**
- 血色屏障也消失了
- 胖子疑惑：为什么骸骨没有追击？
- 江城判断尘然可能埋伏在森林边缘

#### 与尘然遭遇
- 尘然从树后出现，热情招呼："郝兄弟胖兄弟，好久不见"
- 胖子紧张想找武器（尘然一招碾压苏小小）
- 出乎意料：两人并未动手

#### 虚伪的客套
- 江城和尘然**勾肩搭背**，称兄道弟
- 表面亲密，实则互相提防
- **江城的试探**：
  - 告知尘然错过了"好戏"（苏小小牺牲）
  - "下次有白痴我提前通知你"
- **尘然的应对**：
  - 承认错过可惜
  - 主动分享石柱上的线索

#### 互相提防的细节
| 江城 | 尘然 |
|------|------|
| 箍住尘然脖子 | 搂着江城腰部 |
| 手臂肌肉隆起，随时可勒断脖子 | 手背青筋浮现，暗藏暗劲 |
| 拉着尘然一起蹲下看石柱 | 担心胖子"心里不舒服" |

### 第二百一十七章 石画

#### 石柱上的9幅画
- 前8幅清晰，描绘不同场景，彼此关联
- 第9幅是**残图**：
  - 左侧偏上约2/3缺失
  - 右下角只画了一个人
  - 寥寥几笔，幼稚园简笔画水平

#### 胖子发现的关键信息
- 残图中的人影让他想起**老女人**
- 那一瞬间的诡异熟悉感：
  - 蜷缩身体
  - 躲在角落
  - 应该是女人
- 但再看时，熟悉感消失

#### 推测：老女人的真实身份
- **老女人不是玩家，而是NPC**
- 风蚀程度表明石画至少有**几百年历史**
- 老女人在那个时代就已存在
- 她可以**无差别攻击所有玩家**

#### 黑石镇的7人之谜
- 黑石镇默认有**7位"客人"**
- 第一次：医生、胖子、尘然、老女人、机车女、梁龙（6人）
- 第二次：医生、胖子、尘然、梁龙（4人）
- **推测**：镇民通过某种途径知道总人数，包括"非人"的存在
- 所以不为老女人准备食物

#### 上楼前的紧张
- 胖子被江城用手指捅醒（吓到跳起半米）
- 楼梯呈**螺旋形**，中间是深渊
- 两人依旧保持贴身姿态前行
- 胖子暗中盘算：能否联合江城将尘然推下去

## 关键信息

### 江城与尘然的博弈
| 维度 | 江城 | 尘然 |
|------|------|------|
| 表面 | 称兄道弟、热情洋溢 | 配合演戏、笑容可掬 |
| 实际 | 随时准备勒断脖子 | 暗劲藏于腰间，随时发难 |
| 目的 | 拿线索，监视尘然 | 拿线索，拖延时间 |

### 石画的秘密
- 共9幅画，网格排列
- 前8幅：连贯的故事场景
- 第9幅（残图）：右下角有疑似老女人的人影
- **暗示**：老女人可能是这个故事的关键人物

### 老女人的身份线索
| 线索 | 内容 |
|------|------|
| 时间线 | 石画有数百年历史 |
| 身份 | 不是玩家，是特殊NPC |
| 能力 | 可无差别攻击所有玩家 |
| 特征 | 从白带灰变成"无色"，比婴儿还空白 |
| 关联 | 黑石镇的7人位置与她有关 |

### 苏小小牺牲的效果
- 血色屏障消失
- 骸骨没有追击
- 石画内容揭示：**她的死让一切恢复原状**
