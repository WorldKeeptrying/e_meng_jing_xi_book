# 第三章-剧情

## 核心事件：车祸与别墅

### 大巴车到来
- 豪华大巴车停靠站台，车门开启
- **郑导**出现：寸头女，20出头，嚼口香糖，自称导游
- 众人按顺序上车

### 车上人物
**新加入的"旅客"（可能是NPC）**：
- 情侣：女孩看杂志，男孩戴眼罩耳机休息
- 母子：中年女人+大男孩（男孩情绪低落，**手指缠绷带**）

### 座位安排
| 位置 | 人员 |
|------|------|
| 司机后第一排 | 母子 |
| 壮汉 | 母子身后 |
| 唇边痣女人 | 郑导身后 |
| 情侣后第二排 | 江城 + 胖子 |
| 最后一排 | 秃顶中年（脸色阴沉） |
| 某处 | 男文员 + 清纯女（拼座） |

### 江城与胖子的对话（名场面）
- 江城添油加醋转述梦界规则
- 胖子：在这里死掉就会全家倒立流血暴毙？
- 江城：因为我是一个孤儿。

### 车祸发生
- **突发事件**：大巴车急刹车撞树
- **原因**：大雨导致道路一侧坍塌，车辆失控
- **后果**：
  - 玻璃碴飞溅，众人受伤（轻伤）
  - 车辆无法继续行驶
  - 男文员受惊，被壮汉拖出车厢+扇耳光威胁闭嘴

### 任务开始：林中别墅
- 郑导建议众人去林中别墅避雨
- 壮汉等人应允 → **真正任务开始**
- 郑导返回拿雨伞（细节：她胳膊上有血迹）
- 众人步行约半小时到达

### 别墅外观
- 三层别墅，被密林掩盖
- 破败，至少5年无人打理
- 门前堆满落叶

### 关键细节
1. **壮汉情绪变差**：对清纯女的牢骚反应过度
2. **江城嗅到恐惧**：从牢骚与不满中察觉众人的恐惧
3. **门没锁**：厚实木门一推就开
4. **客厅状况**：稍显陈旧，地板和餐桌有薄薄灰尘，不像5年无人

## 悬念
1. 别墅内有什么任务/危险？
2. 男孩手指为什么缠绷带？
3. 秃顶中年为什么脸色阴沉？
4. 壮汉为什么情绪变差？
5. 客厅有灰尘但不破败，说明有人近期来过？
