# 第三十六章 二次发育

## 主要事件

### 食堂探索
- 食堂为独立二层建筑，位于西广场小树林旁
- 一层被屏风分为东西两侧：东侧学生区（面积大）、西侧教职工区（人流稀少）
- 江城发现其他队友未在食堂出现，推测他们刻意避开人群以降低风险

### 早餐插曲
- 江城点了6个硬菜（糖醋排骨、海鲜粥、煎刀鱼等），声称"二次发育"要多吃
- 胖子吐槽他10年前就终止发育了
- 江城用戏谑的方式化解了胖子的紧张情绪

### 江城的核心推断
与胖子共进早餐时，江城提出了对旗袍女之死的关键分析：

1. **旗袍女冒险外出必定有隐情**：她不是新人，应该清楚夜间的危险性
2. **被逼离安全空间的逻辑**：假设她遇到鬼，鬼为何不直接在宿舍动手，而要将她驱赶到走廊？
3. **核心推断**：她确定自己已被鬼锁定，不走必死，离开或许还有一线生机

> "能将我逼出宿舍这个目前看起来比较安全的空间，来到走廊这个看起来就知道不安全的地方，那么只有一种可能——那就是我确定我已经被鬼锁定了。"
