# 第六十八至七十章-剧情

## 第六十八章 搭档

### 余雯和周太福的关系揭露

**真实身份**
- **雇佣兵**：两人同是雇佣兵，相识相知相恋于东南亚雨林的烂泥潭里
- **搭档与恋人**：不仅是搭档，也是恋人
- **雇佣兵生涯**：干着谁出的钱多就为谁卖命的勾当
- **沉默、冷血而高效**：无论是放下武器的士兵，还是老人、孩子，一概不留活口

**雇佣兵生涯细节**
- **最初的约定**：赚够200万就收手，回家乡的小镇上开一家不大不小的超市，过平静安稳的日子
- **逐渐忘却约定**：随着能力与配合愈发纯熟，声名鹊起
- **杀戮中的绽放**：夜里射出的枪弹尾焰划出好看的流线，人命如同草芥
- **疯狂的满足**：极度疯狂后所带来的片刻的满足
- **人心如壑**：这还远远不够

**进入噩梦世界**
- 在睡梦里，推开一扇陌生的门，进入噩梦世界

### 前往档案室

**目标地点**
- **档案室**：前往档案室寻找线索
- **冯兰的信息**：余雯问过冯兰，用了很多说辞才得知档案室的位置

**周太福的质疑**
- "那个冯兰会不会有问题？"
- 余雯回答："不知道，但小心为上"

**时间紧迫**
- **甄建仁拖延**：那个甄建仁恐怕拖不了多长时间
- **避免招来学生老师**：省得到时候再把学生老师招过来
- **周太福的凶光**：她们要是敢来就只能怪她们命不好了
- **余雯的警告**："尽量不要再杀人，这里有这里的规矩"

### 发现档案室

**奇怪房间**
- 房间从外面看占据三四个寻常房间那么大面积
- 门是较新的合金防盗门（刚换不到半年）
- 其他房间不少连门锁都锈蚀了

**血腥气味**
- 二人闻到血的味道
- 里面看来有情况

**合作默契**
- 周太福将余雯挡在身后
- 左手拉门把手，右手手腕一翻出现匕首
- 余雯习惯性帮他护住身后
- 拉开门后立即后撤

### 发现罗一尸体

**尸体状态**
- 趴在地上，头朝外，脚冲里
- 一身打扮是罗一无疑
- 刺鼻的血腥味扑面而来

**余雯的真相洞察**
- "果然是他……"
- 脸上没有丝毫意外表情
- 早在404开会时，便留意到本该由江城保管的相机不见了
- **找到相机**：蹲下系鞋带的功夫，找到被藏于床下的相机
- **察觉江城意图**：想到江城这么做的原因：他们中恐怕混入了鬼

### 余雯的分析过程

**鬼混入的推断**
- 解释了为什么越接近任务结束时限，人死的速度反而慢了下来，甚至鬼还无故消失了

**排除嫌疑人**
- **首先排除江城**，然后胖子也可以排除
- **自己这面的三个人**：也应该可以排除，因为三人始终没有分开，鬼没有杀人后再替换的机会
- **剩下的**：就是405的两个人（甄建仁和罗一）

**观察甄建仁和罗一**
- 看不出什么，两人的行为举止都合乎常理，并无破绽
- 罗一在听到分析鬼的时候，眼神中隐隐流露出恐惧

**与张茵茵的默契**
- 灵机一动看向张茵茵的位置
- 张茵茵感知力可怕，几秒钟后注意到余雯的视线
- 二者的视线仅仅触及刹那，就都不着痕迹的移开

**镜子线索**
- 回到406房间后，余雯用余光发觉张茵茵的手上有一道光掠过（镜子）
- 走过去贴在张茵茵耳侧低声说："你手里的镜子能反射出鬼，昨天我见你用过。现在……鬼已经混进来了"

## 第六十九章 手指

### 镜子确认真相

**通过镜子反射**
- 在镜子的反射中，余雯和张茵茵确定了罗一的身份
- 他是鬼
- 真正的罗一已经被鬼杀掉了

**罗一的死亡时间**
- 是什么时间死的？又死在了哪里？
- **答案**：据昨天甄建仁所说，他先赶来与大家汇合，而罗一一个人去了档案室调查
- 他在那里与鬼遭遇，并被后者杀死，取而代之
- 解释了为什么"罗一"回来后说在档案室什么都没找到

### 档案室内部勘察

**内部环境**
- 面积很大，罗一倒地的位置附近摆着木质办公桌和老式靠椅
- 房间呈狭长型，一排排的深绿色铁架
- 少数铁架上有浅蓝色的笔记夹
- 远处铁架上摆满了密密麻麻的牛皮纸袋

**光线问题**
- 唯一的窗户就在办公桌前方不远处
- 房间深处应该有窗，但被封死或挂上了深色窗帘遮挡
- 熹微的光亮下，里面影影绰绰

**电力故障**
- 找到开关摁下去但没什么反应
- 时间太久，即便有灯也废了

### 罗一尸体详细检查

**死亡姿势**
- 上半身被鲜血浸透，衣领被撕开，身体呈现不自然的扭曲
- **左臂断掉**：从手肘处断掉，被反关节拧在身后
- **挣扎痕迹**：死前似乎还挣扎过

**死亡位置**
- 头朝向门的位置，脚朝着里面
- **结论**：是在即将离开的时候被袭击，倒地后呈现出这样的姿势

**鬼的习惯**
- 下巴不见了（鬼的习惯）
- **特别之处**：左臂是被巨力扭断的，里面的骨头几乎全碎了
- 见过的尸体中只有他遭遇了这样的事

**手指发现**
- 手指也被掰断了！
- 抬起压在身下的右手：除了血污，右臂完好无损，手指也是
- **惯用手**：罗一惯用的好像是左手

### 线索分析

**推测过程**
- 罗一已经找到了有关鬼的线索，正准备带着线索离开的时候，被鬼突然袭击
- 在撕扯中，被鬼捏碎了左臂
- 因为他惯用左手，所以当时他也是左手拿着有关鬼的资料

**鬼拿走资料**
- 资料被鬼拿走，甚至可能已经销毁
- 这条线索到这就算是断了
- 只能去器械室或C座碰运气，危险性高

**余雯的乐观分析**
- "不用那么悲观，我想即便是鬼拿走了资料，大概率也是不会销毁的，我们或许能在其它地方找到"

**规则理论**
- 规则不会允许鬼这么做
- 否则鬼大可以在罗一找到有关她的资料前结果了他，何苦等到他已经拿到了资料，准备离开的时候才下手
- 或者在罗一来之前就销毁掉有关她的线索

**任务规则核心**
- 每个任务中的鬼都要受到任务里规则的限制
- 虽然随着任务时限的临近，鬼会被释放出更多更可怕的能力（封印逐渐解开）
- 但只要没到最后一刻，鬼就不是无敌的
- 鬼最多只是将线索藏得越来越深，越来越隐蔽，却无法从根源上消除线索
- "这是任务规则里为我们留下的最后的生路"

**线索藏匿地点**
- 周太福推测：音乐教室？器械室？还是完全不相干的某个角落？
- **余雯推测**："如果我是鬼的话...我会将它放回原位"

## 第七十章 档案

### 档案室线索搜索

**档案架状况**
- 档案架共分为4层，由上至下
- 除了第一层没有翻动过的痕迹，其它三层都被翻得十分乱
- 地上散落着不少牛皮纸袋

**罗一的活动痕迹**
- 纸袋上没有遗落灰尘，是最近才弄成这样的
- 罗一在这里取走了有关女鬼的线索

**线索缺失**
- 重新找过后，没有任何发现
- **周太福结论**：看来是鬼将线索带走了
- **时间紧迫**：明晚午夜就是最后的时限

**下一步行动**
- 余雯不再犹豫，转身向外走去："走"
- **目标**：C座，音乐教室

### 江城的行动

**隐藏与观察**
- 一阵极其轻微的摩擦声响起，有扇关闭的门被推开了一条缝
- 门位于档案室的对面，毫不起眼
- 门缝处缓缓浮现出一只布满血丝的眼睛（胖子）
- **胖子报告**："医生，他们走了"

**江城现身**
- 江城从沾满灰尘的藤椅上坐起
- 起身后拍了拍屁股上的灰
- 腋下夹着个牛皮纸袋，纸袋一角沾染着血迹

**胖子的钦佩**
- "医生你可真厉害，你怎么知道他们会来档案室找线索？"

**江城的推理**
- "罗一被鬼替换，他们自然会想到他是死在档案室，进而联想到他可能是找到了什么重要线索而被鬼灭口"

### 镜子验证鬼的方式

**胖子的问题**
- "他们没有相机，是怎么确认罗一是鬼假扮的？"

**江城的回答**
- "是镜子，我见张茵茵用过，通过镜子反射也可以验证鬼"

**张茵茵缺席**
- 胖子意识到刚才并没看到张茵茵
- "那个张茵茵和他们分开了"
- 江城点头："嗯"

### 江城的神秘预测

**时间快到了**
- 看向门的位置，视线好似能穿透这扇门，看到外面的景象
- "时间快到了"

**胖子的感受**
- 虽然没听懂江城的话，但本能地觉得这里面夹杂着某些深奥且复杂的问题
- 明白即便问了江城也不会解释
- **江城的特点**：既古怪，又极度自我的男人
- **驱动力**：驱使江城行动的行为准则不是自己能理解的

**路上的恐惧**
- 来的路上，江城攥住胖子衣领，低头俯视他的瞬间
- 胖子甚至觉得前者会杀了自己

### 档案内容

**册子递送**
- 江城随手把册子递了过去
- 册子不大，算上封皮也不厚，上面用烫金字体写着"09级舞蹈系学生档案"

**余雯的推测验证**
- 正如余雯的推测，鬼杀了罗一后，取走了他手上的档案，放回了原处

**江城捷足先登**
- 找路和解决掉张茵茵浪费了余雯和周太福过多的时间
- 导致江城捷足登先，先一步取走了档案
- 听到有人上楼的声音时，躲进了档案室对面的小房间

### 隐藏房间环境

**办公室特征**
- 曾经应该是间办公室，规模不大，设施齐全
- 唯一的一扇窗被深色窗帘遮挡
- 深棕色的书架立在墙边，旁边挨着台白色饮水机
- 饮水机里面的水早已经干涸
- 地上摆着4、5个替换下来的空桶

**其他物品**
- 几个用来养花的红泥盆整整齐齐地码成一排，摆在窗户下
- 里面的植物早已枯萎不成样子
- 胖子身后有个木质衣架，上面挂着几件衣服

**时间线索**
- 老旧的办公桌上放着一本台历
- 江城拿起台历，发现最后的时间停留在09年的12月
- **胖子观察**：这间办公室里的人好像是匆忙间离开的，里面的东西都没有带走

### 学院历史真相

**胖子的问题**
- "怎么，这间学院还有舞蹈系？"
- 记得这里是一间专门的音乐学院，当初带他们进来的老师也是这么介绍的

**江城的回答**
- "育英音乐学院自然是没有舞蹈系，但育英艺术学院有，那是这间学院的前身"
- **档案里有**：在存放档案的地方找到了10年的档案，那时的名字就已经改成了育英音乐学院
- **舞蹈系裁撤**：舞蹈系也被取消了

**真相揭露**
- 学校突然改名，被裁撤掉的舞蹈系，晦涩不明的档案
- 一切似乎都在表明这里隐藏着什么不为人知的秘密
- **胖子结论**："是因为那个女生的死，才引发了这一系列的变故！"
