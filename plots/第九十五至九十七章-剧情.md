# 第九十五至九十七章 剧情总结

## 第九十五章 疑云

### 关键信息揭露

**任务时限真相**
- 江城向胖子解释：第三次彩排是他们查清真相的时限
- 苏郁的时限是周年庆典——陈瑶选择在庆典上公布真相后再杀死他（相比报仇，她更在乎名誉）
- **核心认知**："过去无法改变，我们能影响的仅仅是过程"，无论是否完成任务，苏郁最终都会死在陈瑶手中

**余雯的审讯**
- 余雯在音乐教室内审问苏郁，将其双腿打断
- 她持有周太福临死前留下的陈瑶日记后半部分
- 结合已找到的线索，拼凑出真相：**苏郁是骗子，骗色不成恼羞成怒，编造弥天大谎逼死陈瑶，这也为自己打开了通往地狱的门**

**余雯的悲剧**
- 断了一条手臂，但有某件神秘物品让她在鬼的袭击中存活
- 后悔没有在周太福被鬼盯上时出声提醒
- 核心矛盾：那件东西救了她，却害死了爱人
- 她是少有的**主动走入噩梦的人**

## 第九十六章 生路

### 镜子世界的发现
- 推开音乐教室门，看到整面墙纸被撕下，暴露出一面**巨大的镜子**
- 光从镜子中透出，里面是10年前的舞蹈教室
- 正在进行一场诡异的舞蹈排练，**没有声音**（默声电影般）

### 镜中"演员"名单
| 身份 | 备注 |
|------|------|
| 旗袍女 | 已死 |
| 龙涛 | 已死 |
| 罗一 | 已死 |
| 甄建仁 | 已死 |
| 周太福 | 下巴巨大创口，已死 |
| 张茵茵 | 已死 |
| 陈瑶（领舞） | 10年前的形态 |
| 余雯 | 观众席上 |
| 苏郁 | 浑身浴血，被绑着 |

### 生路触发
- 胖子给苏郁一个脑瓜崩，舞台侧后方凭空出现一扇**铁门**
- "演员"们全部停止动作，背对众人
- **隐藏生路**：查清真相，将苏郁带到曾经的舞蹈教室，交给陈瑶
- 余雯将苏郁丢在陈瑶身后，陈瑶与苏郁一同消失
- 余雯企图独占生路："只要我一个人活着出去就好了"

### 关键发现
- 江城发现镜子外的音乐教室也出现了一扇门
- 镜子里的门是**假的/陷阱**

## 第九十七章 救？

### 胖子与江城的讨论（回忆）
- 若未在规定时间内通过门，会被"不知名的力量抹杀"
- 樊力的遗物招领就是例证（面目全非的尸体）

### 致命陷阱
- 余雯拉开镜中铁门，却发现：
  - 门里站着背对她的**甄建仁**
  - 身后的甄建仁也转过身来
  - 甄建仁倒提拐杖，如同长矛洞穿了余雯的**两只脚踝**
- 甄建仁的动作"机械般反关节旋转，没有声音"
- 余雯惨叫哀嚎，向镜子爬去求救

### 抉择
- 胖子惊觉镜子里的陷阱，意识到只有穿过镜子回到现实的那扇门才是真的
- 江城推了胖子一把：**"你先走，我去救她"**
