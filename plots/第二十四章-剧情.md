# 第二十四章 咖啡 - 剧情总结

## 核心事件
- 江城解释**精力无法恢复**，长此以往会被噩梦耗死
- 江城做咖啡，**速溶涨价到1块5**
- 江城在咖啡里放**安眠药**
- 江城看恐怖片《山村老尸》，胖子不敢看
- 江城科普**楚人美**是"鲁迅笔下《茶花女》原型"（假知识）
- 胖子穿鞋睡觉（江城提醒）
- 江城体质异常（冷热不敏感，极少生病，痛感迟钝）
- 江城再次进入噩梦：**学校场景**

## 关键信息

### 噩梦机制
| 规则 | 描述 |
|------|------|
| 体力恢复 | 噩梦中消耗的体力返回现实后完全恢复 |
| **精力无法恢复** | 耗费的精神力无法恢复 |
| 后果 | 越来越疲惫，最终被噩梦活活耗死 |
| 硬挺不睡 | 最多挺几个晚上，挺不住睡着后进入噩梦必死 |

### 咖啡事件
江城询问胖子喝什么：
| 咖啡 | 有无 |
|------|------|
| 卡布奇诺 | 没有 |
| 拿铁 | 没有 |
| 美式 | 没有 |
| 意式 | 没有 |
| **速溶** | 有（1块5一条，涨价了） |

江城："因为**生活需要仪式感**"
胖子："只有速溶让我选什么？"
江城："这次**安眠药**我没放太多，你先尝尝咸淡"

### 《山村老尸》事件
- 江城叫胖子一起看，胖子死活不去
- 胖子听到诡异唱腔想去关电影，赶上楚人美现身
- 江城科普："楚人美是封建社会苦难女子，鲁迅笔下《茶花女》原型"
- **这是假知识**：鲁迅没写过《茶花女》，楚人美是电影虚构角色
- 胖子自动屏蔽

### 江城体质
| 特征 | 描述 |
|------|------|
| 冷热感知 | 不敏感 |
| 生病 | 极少 |
| 痛感神经 | 迟钝 |
| 自我感觉 | "没有感情的工具人" |
| 饭量 | 极大（吃大半锅排骨+两碗饭） |

### 胖子行为
- 穿鞋睡觉（江城提醒，刷了旧鞋+买了新运动鞋）
- 怕恐怖片，堵住耳朵
- 想和江城一起睡（被关门拒绝）
- 昨晚觉得冷（江城给新被子）

### 新噩梦场景
| 属性 | 描述 |
|------|------|
| 入口 | 黑色铁门出现在一楼拐角处 |
| 推门 | 身体先于大脑发号施令 |
| 眩晕 | 诡异眩晕感 |
| 场景 | **类似学校**，规模不大 |
| 时间 | 下午4-5点，夕阳 |
| 建筑 | 几栋，深灰色影子 |
| 大门 | 已关闭 |
| 其他人 | 附近暂时没看到 |
| 江城行动 | 果断转身朝反方向走去 |

### 进入噩梦前兆
- 耳边回荡莫名旋律
- 不是古典、流行、爵士
- "生于黑暗，归于黑暗"
- 沉入大洋深处/走入没有尽头的隧道
- 绝望，只有绝望
- 清脆破裂声（纯黑色布帛被撕开）
- 猛然惊醒，站在铁门前

## 细节

### 江城性格
- **生活仪式感**：速溶咖啡也要选
- **幽默/假知识**：楚人美=鲁迅《茶花女》
- **安眠药**：放咖啡里，"没放太多"
- **恐怖片**：吃着薯条蘸番茄酱看
- **差点睡着**：看完恐怖片
- **关门反锁**：习惯动作
- **果断**：进入新场景后朝反方向走

### 悬念
1. 江城买药与新噩梦有关？
2. 精力无法恢复 = 最终会被耗死
3. 安眠药是帮助进入还是防止进入？
4. 学校场景是什么？
5. 江城体质为何异常？
