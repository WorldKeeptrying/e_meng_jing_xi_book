# 第二章-剧情

## 核心事件：梦界集合

### 场景：废弃汽车站
- 天气：阴森，飘雨，灰暗色调
- 环境：破败，露出红砖的站台，老旧失修的遮雨板
- 人数：5人先到达 + 1人（胖子）后到 = 共6人

### 人物集合
| 角色 | 特征 | 备注 |
|------|------|------|
| 江城 | 工装外套+格子睡裤+卡通毛绒拖鞋 | 新人，异常冷静 |
| 清纯女 | 身材娇小，模样清纯 | 善良，借胖子围巾 |
| 络腮胡壮汉 | 体型壮硕，满脸络腮胡 | 资深者，解释规则 |
| 唇边痣女人 | 35岁，薄唇，唇边有痣 | 资深者，对江城感兴趣 |
| 秃顶中年 | 50岁上下，秃顶，猥琐 | 状况外 |
| 文员男 | 年轻，一丝不苟，文员打扮 | 刚哭过，佝偻 |
| 胖子 | 胖但灵活 | 与江城有交易 |

### 梦界规则（壮汉解释）
1. **进入方式**：通过梦中出现的一扇不存在的黑色铁门
2. **任务机制**：根据场景找出线索完成任务，尽力活下去
3. **场景随机**：可能是村庄、高楼、森林、雪原、荒野等任何现实场景
4. **死亡后果**：在梦界死亡 = 现实世界被抹除（失踪）
5. **离开方式**：完成任务后可回到现实世界
6. **诡异现象**：可能在门后世界遇到自己（有两个人一模一样的情况）

### 江城的表现
- **超强推理**：
  - 推断出壮汉在等人
  - 准确说出等的是一个"胖子，很灵活"
  - **揭秘**：路上遇到胖子，替他先侦查，代价是大衣+雨伞
- **冷静异常**：出现在陌生环境毫不慌张

### 关键细节
- 江城睡觉时穿的是睡衣（格子睡裤+卡通毛绒拖鞋）
- 工装外套明显大几个尺码（可能不是自己的）
- 任务因胖子未到而迟迟未开启
- 世界天色渐暗，雨越来越大

## 悬念
1. 这次任务是什么？
2. 其他新人的状态（文员为什么哭过？）
3. 梦界的本质是什么？
4. 江城如何推理出胖子的特征？
