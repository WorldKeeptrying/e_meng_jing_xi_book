# 第八十六至八十八章-剧情

## 第八十六章 木门

### 前往器械中心
- 江城预言"天不会再亮了"
- 两人快步向西边的器械中心走去
- 校园陷入死寂，秋虫的悲鸣销声匿迹

### 红灯笼
- 远远望见器械中心院落
- **门前挂着两只大红灯笼**
- 散发出猩红色的光，诡异非常
- 胖子吓得躲在树后

### 发现偏门
- 正门被巨大铁锁和铁链从外锁死
- 胖子疑惑：锁怎么是从外面锁上的？
- 在隐蔽处发现一扇**木门**
- **门被木板从外封死，钉满巨大钉子**
- 门缝里塞满黄色纸符，折成三角形

### 道家符篆
- 胖子发现一张黄色纸
- 用鲜红色染料画着不明图案
- 旁边有类似文字的东西（不认识）
- **江城判断**：这是道家的符篆，用来震慑鬼

### 新的推测
- 胖子猜测：陈瑶死后要和苏郁人鬼情未了，苏郁求来符咒自保
- **江城质疑**：
  - 苏郁为什么不离开学校？
  - 器械中心改造是大工程，管理员点头不够
  - **暗示上层默许——李妍薇**
- 门内传出"嘶嘶"类似蛇吐信的声音

## 第八十七章 窥

### 鬼在门后
- 门后有东西在窥视
- 胖子吓得要回去
- 江城决定留下：**"天不会再亮了，没有明天"**
- 胖子选择跟随江城

### 锁孔窥视
- 胖子透过锁孔向院内看
- 起初看到白蒙蒙的浓雾
- 浓雾中有**密密麻麻的红线**
- **浓雾退却后真相揭露**：
  - 那不是雾
  - 是**鬼的眼睛**——乳白浊黄，布满血丝
- 鬼之前趴在锁孔处向院外窥视，和胖子姿势一样！

### 苏郁现身
- 江城推开门缝
- 看清门后家伙的真容：
  - **苏郁**：瘸着一条腿，用拐杖
  - 头向一侧歪斜，身体不协调
  - 衣服老旧，像尘封的布娃娃
- **可怕的眼睛**：
  - 黑色瞳孔几乎消失
  - 只有乳黄色浑浊眼白
  - 布满血丝，藏着无尽怨念
- **关键疑问**：苏郁他……才是真正的鬼？！

## 第八十八章 铁骨铮铮苏郁

### 制服苏郁
- 苏郁钻出门缝，伸手要抓胖子
- 江城用**两个红色花盆**砸晕苏郁
- 胖子被吓昏

### 审问苏郁
- 苏郁醒来后绑在椅子上
- 态度凶狠，称他们为"可耻的帮凶"
- 提及高频词：帮凶、该死、变态、占有、陈瑶、**李妍薇**
- 声称"陈瑶会来找你们"

### 江城的真相揭露
- 江城表明立场：和他们不是一伙，要查清当年真相
- 单独审问苏郁后，苏郁全盘招供
- **苏郁将陈瑶的日记交出**

### 真相：三角关系
- **陈瑶与苏郁**：**自由恋爱**
- **李妍薇**：从中作梗，利用校长女儿身份作威作福
- **李妍薇的罪行**：
  - 强行介入陈瑶和苏郁的感情
  - 精神有问题，占有欲极强
  - 陈瑶因不按她的要求分手，被**诬陷成放荡的女人**

### 关键证据
- 玻璃下压着照片：陈瑶与苏郁的合影
- 两人穿舞蹈服，动作优美，双宿双飞
- **证明他们是恋人关系**

## 重大剧情转折

### 李妍薇的谎言被揭穿
| 李妍薇的说辞 | 真相 |
|-------------|------|
| 陈瑶勾引苏郁 | 陈瑶和苏郁是自由恋爱 |
| 她是受害者 | 她才是加害者 |
| 陈瑶放荡 | 陈瑶被诬陷 |

### 人物关系重塑
```
           李妍薇（加害者）
               │
           强行介入/诬陷
               │
               ↓
    陈瑶 ←──────→ 苏郁
    （受害者）    （自由恋爱）
```

### 新悬念
1. 苏郁的身体为何呈现鬼的特征？
2. 李妍薇还会如何反击？
3. 陈瑶的日记里还有什么秘密？
4. "天不会再亮了"预示什么？
