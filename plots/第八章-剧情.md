# 第八章 - 剧情总结

## 核心事件

### 线索分析

**清纯女推理**：
- 浴缸里的女人长发可能属于受虐待的女孩
- 推理：女孩受尽虐待而死 → 怨气不消化作厉鬼 → 凌晨杀死中年男人
- 目前看来这个分析还算合理

**壮汉状态**：
- 从卫生间出来后气势弱了许多
- 额头有细密冷汗（内心恐惧）

### 身份揭露

众人围坐吃饭时互相介绍：

| 角色 | 真名 | 职业 | 备注 |
|------|------|------|------|
| 壮汉 | 樊力 | 保镖 | 有经验，团队领导者 |
| 男文员 | 谢雨 | 保险公司 | 胆小，情绪化 |
| 清纯女 | 陈晓萌 | 大学生（法学） | 有故事，对虐待话题有执念 |
| 唇边痣女人 | 暖姐 | 卖货（疑似特殊行业） | 气场强大，年龄最大 |
| 江城 | **郝帅**（假名） | **男模**（KTV上夜班） | 自称，真实性存疑 |

### 江城身份疑点

**自称**：
- 姓名：郝帅
- 职业：男模，在KTV上夜班

**疑点**：
- 之前表现得像心理医生/咨询师
- 分析能力、冷静态度与"男模"身份不符
- 用"似有深意的眼神"回应清纯女
- 可能是假名/假身份

### 时间异常

**白天短暂**：
- 几人聊天后天色再度暗下来
- 樊力观察：不是要下雨，是天真的黑了
- 半小时后夜幕降临

**暗示**：
- 噩梦结界中时间流逝异常
- 白天时间被压缩

### 住宿安排

**二层分配**：
- **主卧室**：暖姐 + 陈晓萌 + 樊力（有经验者）
- **男孩卧室**：江城 + 胖子 + 谢雨（后来加入）

**谢雨的恐惧**：
- 被孤立后害怕，求江城开门
- 之前要将江城一人扔进卫生间，现在道歉
- 江城用"郝帅"名字测试他是否真诚
- 谢雨反复叫"郝帅"获得原谅

### 江城行为异常

**不睡床**：
- 选择睡地板
- 理由：睡不惯，在家就睡地板，得劲
- 谢雨猜测：嫂子凶你？

**可能的原因**：
- 职业习惯（心理师的谨慎？）
- 保持警惕
- 地板更"得劲"可能是借口

### 角色心理

**陈晓萌**：
- 对"女孩受虐待"理论有执念
- 每次说到这情绪都变得激动
- 与之前温婉模样大相径庭
- 江城推测：法律专业，或许是有故事的女孩

**暖姐**：
- 说"也许会错过什么"（拒绝谢雨同住请求）
- 暗示：可能发生重要事件需要见证
- 气场强大，不是普通商贩

## 新增信息

- **时间压缩**：白天短暂，黑夜快速降临
- **安全考量**：为避免去一层（有尸体），都睡二层
