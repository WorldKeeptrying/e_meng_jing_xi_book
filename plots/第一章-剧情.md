# 第一章-剧情

## 核心事件：噩梦开始

### 故事引入
- 胡燕（47岁）失踪，深夜穿着浅粉色丝绸睡衣离家
- 失踪前曾找心理医生江城咨询，留下录音

### 录音内容（胡燕的陈述）
1. **起因**：三天前夜里接到妹妹电话
2. **妹妹的异常**：牙齿打颤，声音颤抖，重复讲述一个梦
3. **梦境内容**：
   - 夜里醒来
   - 离开床，从阁楼沿楼梯走下
   - 停在**地下室的门前**
   - 感觉极其真实
   - **关键矛盾**：她家根本没有地下室！
4. **妹妹的结果**：失踪（在打电话那夜）
5. **胡燕的遭遇**：昨天也梦到了同一扇门

### 警方调查
- 一男一女两名警官来到江城工作室
- 根据线索追踪到胡燕曾来过这里
- 江城如实陈述并交出录音
- 警方暗示怀疑胡燕失踪与亲属有关

### 江城的分析
1. 胡燕描述完整，细节到位，不同于一般妄想症患者
2. 胡燕妹妹失踪可信（与警方怀疑相互印证）
3. 胡燕深夜穿薄睡衣离家不符合常理
4. 结论：事件不简单

### 转折：江城中招
- 江城入睡后突然惊醒
- **身体不受控制**，机械地：
  - 起床 → 走出卧室 → 经过客厅 → 沿楼梯下楼
  - 全程无声（楼梯没发出声音，环境死寂）
- 最终停在**一扇纯黑色的、嵌在墙壁里的门前**

## 关键悬念
1. 那扇门是什么？
2. 为什么梦见门的人会失踪？
3. 江城的身体为何不受控制？
4. 门后面有什么？

## 世界观线索
- **噩梦世界**存在，会将人拉入其中
- 在梦中出现一扇门是前兆
- 失踪者可能被拉入噩梦世界
