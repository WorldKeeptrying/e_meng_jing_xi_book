# 第一百六十九至一百七十章 剧情总结

## 第一百六十九章 夺魁

### 关键事件

1. **神秘女孩追查江城**
   - 女孩打电话给手下，排查京地附近十几家精神病院最近一年内康复出院的病人
   - 未发现目标人物
   - 女孩要求扩大搜索范围，并透露目标身边很可能跟着一个胖子
   - 特别强调：这个胖子特别灵活，与躯体严重不符的协调能力
   - 手下敷衍应对，觉得"胖子灵活"算不上什么特别特征

2. **女孩的私密空间**
   - 独自住在临湖的大卧室，落地窗正对湛蓝湖水
   - 穿酒红色睡衣，白皙几乎透明的肌肤
   - 与某人形容的一样，是"一具并不玲珑有致的娇躯"
   - 散乱头发扎马尾，赤脚走到窗前

3. **女孩的素描画**
   - 房间内支着画架，实木地板铺着动物毛皮
   - 画上是素描：简单笔触勾勒出一个年轻男人
   - 笔触如雕塑般精准，寥寥几笔神韵跃然纸上
   - 模样出色，"富婆爱不释手的那种类型"
   - 女孩逆光盯画良久，眼神逐渐变冰冷
   - 说道："尹先生，你可要藏好了"

4. **胖子买菜日常**
   - 江城打喷嚏（暗示被人念叨/惦记）
   - 胖子买了小海鲜、鱼、排骨、小青菜
   - 买鱼经历：老板主动把半死不活的鱼当死的卖，说"它只是条鱼，不该经历这么多的"
   - 胖子手握江城给的假10块钱，想找个地方丢掉以免害人

5. **跑车出现在工作室**
   - 一辆跑车停在工作室前的私人停车位
   - 胖子严格执行江城定的规矩：私人停车位不许停车
   - 剪刀门扬起，走下一个旗袍女人

6. **旗袍女人登场**
   - 浅色旗袍，眉眼如黛，凹凸有致
   - 长发挽成发髻，洛云钗插在发髻中
   - 身上无一丝媚骨，淡然从容
   - 胖子形容："卧槽！"
   - 女人无视胖子，直接推门进入工作室，随意如回自己家

## 第一百七十章 林老板

### 关键事件

1. **尴尬的见面**
   - 旗袍女人进门看到：江城正搂着一位50多岁阿姨，拍背安抚
   - 阿姨哭到动情处紧拉江城不放
   - 江城看到旗袍女人后明显表情变化，想松开但阿姨不放
   - 场面一度尴尬

2. **送走老阿姨**
   - 含情脉脉地离开
   - 江城确认："你……刚来？"
   - 旗袍女人淡定回答："来了有一会了，看你忙着，就没打扰你"
   - 胖子注意到口吻"怪怪的"

3. **胖子的误解**
   - 胖子认定旗袍美女是医生的女朋友
   - 主动躲进厨房，忙活近一小时不给两人创造二人空间
   - 甚至忍住不上厕所

4. **财务汇报场景**
   - 旗袍女人坐在江城办公位置，江城拉椅子坐对面
   - 办公桌上整整齐齐排列着几沓钱
   - 江城主动坦白：本月收入不如上月
   - 原因：几个富婆客户组团去新马泰旅游
   - 旗袍女人面无表情记录客户姓名：钱真美、孙有财、方……
   - 完全公事公办的态度

5. **林老板身份确认**
   - 胖子想起皮阮之前问"林老板来了吗"
   - 旗袍女人就是**林老板**——江城的房东兼老板
   - 胖子之前在工作室发现的女人旗袍，原来是林老板留下的
   - 胖子最初误以为是女朋友，后发现气质与旗袍完全匹配
   - 江城这么抠门的人过得这么惨，说明关系绝非单纯雇佣

6. **冷清的晚餐**
   - 胖子只拿了两副碗筷（没想到自己也算）
   - 整顿饭三人无话，堪称胖子生平最冷清的一顿
   - 江城和林老板默契如同一个人

7. **胖子的胡思乱想**
   - 刷碗时揣测江城与林老板的关系
   - 联想第一次噩梦：别墅中变态囚禁的一家人
   - 荒谬念头：医生是否被林老板强行囚禁？
   - 又想可能是交易关系
   - 想到林老板留在这里的贴身衣物，喉咙滚动

8. **就寝安排**
   - 江城叫胖子上楼
   - 胖子以为要把卧室让出来，主动说去楼下睡
   - 江城说：**都去楼下睡，把卧室空出来**
   - 理由："我最近颈椎不大好，想试试沙发，你睡地板就可以了，凉快"
   - 暗示：林老板住楼上卧室

### 本章要点

- 神秘女孩在追查"尹先生"（江城在梦界中的化名），特别关注身边"灵活的胖子"
- 女孩画了江城的素描，画工精湛，眼神冰冷说"藏好了"
- **林老板正式登场**：绝美旗袍女人，江城的房东兼老板
- 林老板与江城的关系复杂：老板查账、财务汇报，但又留有贴身衣物在工作室
- 江城在林老板面前态度明显转变，不再是日常的毒舌随意
- 胖子从误解（女朋友）到确认（林老板），仍对二人关系充满好奇和想象
- 江城和林老板默契度极高，但保持公事公办的表面距离

## 第一百七十一章 裂开

### 关键事件

1. **林婉儿留人**
   - 江城想带胖子赶紧下楼，林婉儿平淡叫住："他可以走，你留下"
   - 胖子误以为两人要"谈大生意"，老脸通红
   - 江城心不甘情不愿走回去，一脸"上刑场"的大义凛然
   - 胖子脑补医生体力不行（"人越缺乏哪方面就越爱炫耀"）
   - 胖子下一秒眼前一黑，被林婉儿弄昏

2. **第二天醒来**
   - 胖子睡在楼下沙发上，头晕看东西蒙雾气
   - 跑车已离开，以为两人昨晚在楼上卧室
   - 脑海闪过无数马赛克

3. **卧室真相**
   - 江城刚从床垫坐起，上身赤裸
   - 被子下压着林婉儿的淡色旗袍
   - 江城随手将旗袍卷起丢进衣框篮——里面类似的旗袍还有很多
   - 胖子"彻底裂开了"（人也可以细胞分裂）

4. **脖子歪了**
   - 江城脖子向右歪，微微肿起（像落枕）
   - 解释：林婉儿把他弄昏了，"但也仅仅是弄昏而已"
   - 胖子脖子也歪了，肿得转不回去，比落枕疼得多
   - 两人一个向左歪一个向右歪，相互搀扶下楼

5. **过期药**
   - 江城从堆积如山的药盒中找出清凉镇痛贴
   - 胖子体格大一贴不够，江城不情愿地多贴了一贴
   - 江城自己没贴："没过期的在另一个袋子里"

## 第一百七十二章 新图

### 关键事件

1. **林婉儿名字揭晓**
   - 林老板全名：**林婉儿**
   - 胖子问会不会回来，江城："不会，一般不会"
   - 江城明显在逃避有关林婉儿的话题

2. **噩梦收益理论**
   - 江城提出核心问题："我们能从噩梦中得到什么？"
   - 白纸和报纸只能作用于噩梦任务中，获取难度不低却只有一次使用机会——太苛刻
   - 如果所有收益只在噩梦中有效，那为什么要主动下本？
   - **存在即合理**：主动进入噩梦的人一定有匹配风险的目的

3. **胖子推理**
   - 猜测有人搜集报纸/白纸提供给有需要的富人
   - 江城认可"有这种可能，但不会是全部"
   - 江城还猜测：自主进入可能降低难度或延长下次噩梦到来时间
   - 但他认为这些都只是边缘，尚未触及真正核心

4. **日常困境**
   - 胖子把酱油当醋，江城气得睡不好
   - 钱被林婉儿搜走（胖子口袋里的红钞票也没了）
   - 冰箱速冻食品保质期磨掉→江城："买的时候就过期了"
   - 林婉儿开工资但要等几天
   - 江城找皮阮求援，一声令下酒店后厨直接搬来食材

5. **进入新噩梦**
   - 江城梦到自己站起，沿楼梯下楼，漆黑的门出现
   - 光影变换后，身处空旷建筑内
   - 耳畔是一浪高过一浪的波涛声

## 第一百七十三章 潜伏

### 关键事件

1. **海边建筑**
   - 江城判断：体量巨大的波涛声+咸腥味=海边
   - 建筑特征：穹顶极高，四周陡峭岩壁状墙壁（有人工雕琢痕迹）
   - 大理石块裸露，布满倒塌石柱（大小不一，有些两人合抱）
   - 碎石遍布，一动就发出声响

2. **黑雾环境**
   - 非漆黑一片，是稀薄黑色雾气导致
   - 挥手可驱散部分黑雾
   - 石柱上密密麻麻雕刻看不懂的图案

3. **神秘石雕**
   - 顶端折断的石柱上刻着一种生物
   - 有鱼的鳞片，又生有双翼，身后有类似长矛的尖锐尾巴
   - 风蚀水浸使雕刻丧失原貌，只剩模糊轮廓
   - 江城不认识这种生物

4. **伏低隐藏**
   - 江城直觉建筑内不止他一人
   - 无论是敌是友，准备做最后一个现身的人
   - 利用碎石和黑暗保持隐蔽

5. **胖子等人推门**
   - 胖子推门时装腔作势（"这门好沉啊"），实际很怂
   - 胖子看到江城后想叫"医生"，但及时改口装作陌生人
   - 逆光中江城缓缓起身暴露

6. **公子哥暗中潜伏**
   - 江城刚走一步，听到身后异响
   - 身后石柱边站着一位眉清目秀的年轻人
   - 年纪与江城相仿甚至更小，额前细碎长发
   - 温润如玉，彬彬有礼，如同贵族公子哥
   - 身穿熨帖合身的古装，灵动飘逸
   - **关键**：江城之前观察过身后石柱，当时没有人
   - 公子哥是趁海浪声掩护，一步步偷偷摸到江城身后的
   - 说明此人行动能力极强，绝非普通人

7. **8人团队**
   - 4男4女，共8人
   - 其中一个身高超过2米的男人，站在萝莉打扮女孩身后
   - 女孩身高只到2米男人的腰部
   - 黑瘦女人试图攀爬峭壁失败，浑身擦痕渗血
   - 建筑位于海岸边约100米，周围无其他建筑
   - 一侧陡峭岩壁，一面一望无际的森林
   - 离开方式：攀峭壁/穿越森林/造船出海

### 本章要点

- **林婉儿真实面目**：能无声无息弄昏两人，江城习以为常（"一个月一次，月底查账时来"）
- 江城对林婉儿话题明显逃避，关系远比表面复杂
- **噩梦收益论**：主动下本者必有匹配风险的目的，报纸/白纸只是冰山一角
- **新噩梦场景**：海边古怪建筑，黑雾弥漫，神秘石雕生物，8人团队
- **公子哥**：暗藏实力的危险人物，能无声接近江城而不被发现
- 胖子越来越成熟：知道在队友面前隐藏与江城的关系
