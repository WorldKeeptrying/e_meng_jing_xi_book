# 第二十二章 旗袍 - 剧情总结

## 核心事件
- 胖子继续修理工作室（立式衣架）
- 胖子发现**蓝色旗袍**挂在衣架上
- 江城抱出**大桶女人衣服**
- 江城以为少了粉色贴身和白色流苏披肩
- 江城解释衣服是**房东/老板**的
- 胖子三观再次被碾碎
- 胖子心理活动：从误会到震撼到迷茫

## 关键信息

### 旗袍细节
| 属性 | 描述 |
|------|------|
| 颜色 | 蓝色 |
| 款式 | 立领，短袖，真丝材质 |
| 工艺 | 做工精良，牡丹刺绣 |
| 特征 | 大腿上岔开很大 |
| 价格 | 不菲 |
| 气味 | 淡雅香味 |
| 归属 | **房东/老板** |

### 江城缺少的衣服
江城说应该还有：
- 粉色贴身的
- 白色流苏披肩

### 桶里的衣服
- 都是**女人衣服**
- 各式各样
- 旗袍只是其中一件
- 数量多到桶溢出来

### 胖子心理变化
| 阶段 | 想法 | 反应 |
|------|------|------|
| 发现旗袍 | 窥探秘密的快感 | 兴奋，比划 |
| 看到桶里衣服 | 三观碾成粉末 | 瞳孔放大 |
| 误会江城 | "以身饲虎" | 拼不回的三观 |
| 江城解释 | 房东的 | 世界都不好 |
| 出门抽烟 | 迷茫 | 眼神空旷 |

### 房东/老板
| 属性 | 描述 |
|------|------|
| 性别 | 女 |
| 身份 | 江城房东 + 老板 |
| 衣服 | 大量女人衣服留在这里 |
| 关系 | 与江城关系密切？ |

### 江城解释
- "这些衣服并不是你想象中的那些人的"
- "我不是皮阮，我有我的原则"
- "是我房东的，她同时也是我的老板"
- 缺少两件衣服，砸嘴离开

### 胖子最终反应
- 出门抽烟
- "忧郁又迷茫的半张侧脸"
- "写满了故事"
- 想说些什么却没说出口

### 江城行为
- 抱大桶衣服下楼
- 倒出衣服让胖子震惊
- 收拾衣服去洗衣机洗
- 取走蓝色旗袍
- 找另外两件没找到
- 洗衣机声音沉闷（衣服太多或老旧）
- 踢空桶到墙角（不挡楼梯）

## 细节

### 厨房情况
- **没有菜刀**（江城从不下厨）
- 只有钝的水果刀
- 土豆丝变土豆块

### 胖子修理工作
- 立式衣架松动螺丝拧紧
- 接口处锤子敲紧密
- 缠黑色绝缘胶带
- 摇晃试验

### 悬念
1. 房东是谁？为什么衣服留在这里？
2. 缺少的两件衣服去哪了？
3. 江城和房东什么关系？
4. 胖子想说什么没说出口？
5. 江城上楼"神神秘秘"忙什么？
