# 第一百六十五至一百六十六章 剧情总结

## 第一百六十五章 原因

### 关键事件

1. **江城与陈晓萌的真相**
   - 陈晓萌问的是之前报纸的事
   - 她意识到自己配不上江城后想翻脸要回报纸
   - 江城骗她说报纸被胖子抢走了
   - 胖子得知后无语

2. **午饭冷战**
   - 胖子将每道菜公平分成两份
   - 辣子鸡：江城面前都是辣椒，胖子面前都是鸡肉
   - 糖醋鱼：鱼头归江城，鱼身归胖子
   - 江城筷子伸向胖子的盘子，胖子直接把肉扣自己碗里拌着吃
   - 胖子声称自己在黄鹤楼是刷马桶的保洁

3. **噩梦四大规则总结**
   - **规则一：门后的世界**
     - 每扇门后是完全不同的世界
     - 世界是真实存在的历史碎片
     - 无论做什么都无法改变结局，只能改变过程

   - **规则二：选拔机制**
     - 噩梦是放大化的选拔机制，有奖惩措施
     - 全部队友死亡后，最后一人可得空白纸
     - 空白纸可抵挡鬼的伤害，但仅一次，使用后消失
     - 余雯就是靠空白纸活下来的

   - **规则三：报纸与白纸的关系**
     - 报纸优先级低于白纸（白纸能保命）
     - 但功能不同：白纸保命，报纸提供线索
     - 普通人应该选报纸活下去的可能性更大
     - 对高玩来说，白纸+实力=无敌

   - **规则四：脑电波理论**
     - 思考噩梦的形成机理和吸纳玩家的方式
     - 胖子的话打开了新思路
     - 胡燕录音中的杂音只在对方开口时出现

4. **陈晓萌的分析**
   - 她在钱家废楼能从鬼手中逃脱不是运气
   - 手里应该有保命白纸，已经用掉了
   - 所以搜身时只找到合影相片，没有白纸

## 第一百六十六章 红糖水

### 关键事件

1. **江城身体不适**
   - 中午休息不好，用脑过度后遗症
   - 脸色苍白，浑身发抖，用不上力气
   - 胖子十分担心，想送他去医院

2. **江城装病**
   - 胖子去找东西时，江城立刻不抖了
   - 脸色也红润了，眼睛洋溢着快乐又鸡贼的光芒
   - 等胖子端碗回来又装作虚弱

3. **江城点菜**
   - 假装好转后开始点菜：葱爆海参、油焖大虾、干锅鸭头、麻辣小龙虾、烧牛蛙、蜀香五椒鸡、椒盐……
   - 最后说要小白菜排骨汤解腻
   - 胖子算了算，建议还是去医院

4. **皮阮来访**
   - 打扮古怪：白袍子+超级大墨镜
   - 胖子以为是录视频的或病人
   - 皮阮解释是沙特风格的长袍，被客户送的，他又找裁缝改了改
   - 江城吐槽：女朋友死了要穿一身红
   - 皮阮是个难得的好脾气，哈哈一笑过去

### 本章要点
- 江城总结噩梦四大规则，是重要的世界观揭示
- 空白纸是保命道具，报纸是线索道具
- 日常搞笑继续：胖子冷战、江城装病整胖子
- 新角色皮阮出场，是个好脾气的人
