# 第四章-剧情

## 核心事件：别墅探索与噩梦结界规则

### 别墅布局
**一层**：
- 客厅：奢华沙发、毛皮地毯、壁炉（有灰烬）、水晶吊灯（**有电！**）
- 餐厅：冰箱里有大量新鲜食物（蔬菜、米面、肉类、零食、鸡蛋）
- 厨房/水龙头：有干净的水流出
- 储藏室：大量码放整齐的木柴

**二层**：
- 1间主卧室
- 1间书房
- 2间小卧室
- 1间**上了锁的房间**（壮汉和女人都感兴趣，但没打算破门）

**三层**：
- 被黑色铁门锁住（壮汉晃不动）

### 异常现象
1. **有电**：水晶吊灯亮着
2. **食物新鲜**：像一小时前放的，但地上灰尘表明很久无人
3. **有水**：水龙头流出干净水
4. **木柴整齐**：储藏室木柴码得整整齐齐

### 人员分工
- **做饭**：唇边痣女人、江城、胖子（胖子厨艺精湛）
- **取木柴生火**：壮汉、秃顶中年
- **收拾屋子**：清纯女、男文员

### 就寝分歧
- **壮汉主张**：睡客厅（安全）
- **女人主张**：睡二层卧室（可能有线索）
- **结果**：女人+清纯女+男文员上二楼；壮汉+江城+胖子+秃顶中年留客厅

### 守夜安排
- 怀表计时：晚上10点→早上6点，共8小时
- 4人轮流，每人2小时
- 壮汉守第一班

### 噩梦结界规则（重要！）
| 规则 | 说明 |
|------|------|
| **全称** | 噩梦结界 |
| **本质** | 隔绝现实与虚幻的结界墙 |
| **进入** | 一次又一次被卷进来，除非"终结" |
| **受伤** | 只要还有一口气，任务结束后完好无损回到现实 |
| **钟表** | 带进梦界的钟表时间会被打乱，只能计时 |
| **异常** | 违背科学常识的情况都是正常的 |
| **离开** | 找到异常隐藏的线索，破解任务 |

### 江城的推理（时间推断）
- 通过**达棘园小面包**包装上的生产日期：**2006年3月18日**
- 保质期6个月
- 面包口感松软，未受潮（冰箱保存）
- 推断时间：**2006年3月末到9月中某一天**，可能更早

### 壮汉的补充
- 他只是"暂时逃离"，不是终结
- 不知道如何彻底脱离噩梦结界
- 无法弄清楚梦界的时间点
