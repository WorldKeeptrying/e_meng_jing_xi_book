# 第二十章 江医生 - 剧情总结

## 核心事件
- 胖子早起修理江城的**办公室/病例架**
- 胖子做**红烧排骨**，江城大赞但嘴硬说"一般"
- 江城接电话，有**客户**要来
- 胖子主动离开，江城让他去**超市买食材**（晚上做炖排骨）
- 客户**皮阮**到来，穿着名牌，态度谄媚
- 皮阮是为了**女朋友的事**来找江城
- 江城翻阅**胡燕的病例**

## 关键信息

### 办公室修理
胖子趁江城睡觉时修理了办公室：
| 修理项目 | 具体内容 |
|----------|----------|
| 病例架 | 清除宽胶带，钉长钉子固定断裂处，清除锈迹 |
| 电线 | 分开整理，固定到正确位置 |
| 工具 | 锤子、钝头剪刀、黑色绝缘胶带、金属钉 |
| 折叠梯 | 垫纸靠墙放置（防止弄脏墙） |

### 江城与胖子互动
| 事件 | 江城表现 | 胖子反应 |
|------|----------|----------|
| 胖子做红烧排骨 | 吃大半锅+两碗饭 | 满怀期待求夸奖 |
| 评价菜品 | "一般"（嘴硬） | 眼中光消散 |
| 说手纸贵 | 毒舌 | 感动被泼冷水 |
| 让买食材 | 大言不惭指挥 | 激动答应 |

### 皮阮（新客户）
| 属性 | 描述 |
|------|------|
| 外貌 | 全套阿玛尼西装，玳瑁框眼镜，烫金领口衬衫，蓝白条纹领带，棕色皮鞋 |
| 发型 | 中分（像抗战电影带路党） |
| 性格 | 自带气场但一笑毁所有，满脸谄媚 |
| 来意 | **女朋友的事**（不是第一次来找江城） |
| 经济状况 | 富裕（全套名牌） |

### 胡燕病例内容
- 记录较为**详细**
- 包含胡燕**抖腿**动作
- 交流中**视线多次向门的方向看**（极为警惕）
- 胡燕梦到一扇门后失踪
- 妹妹先失踪，电话告知胡燕
- 胡燕面对面告知江城

### 江城心理医生身份确认
- 有**工作室/办公室**
- 存放**患者病例**
- 有**客户**预约来访
- 称呼为"**江医生**"
- 皮阮不是第一次来（"还是你女朋友的事？"）

## 细节

### 江城性格
- **嘴硬心软**：让胖子留下但装作不在意
- **贪吃**：一人吃大半锅排骨
- **毒舌**：评价"一般"，嫌弃手纸
- **青春期长痘**：自称（20多岁的心理医生还青春期？）
- **喜欢指挥人**：让胖子去买菜
- **观察力强**：记录患者微表情和动作

### 胖子性格
- **勤快**：主动修理办公室，做饭洗碗
- **厨艺精湛**：醉仙楼学徒经历
- **容易感动**：江城关心手就被感动
- **渴望认可**：期待夸奖像没得到大人夸奖的孩子
- **忠诚**：被支使去买菜也高兴

### 悬念
1. 皮阮女朋友的什么问题？（可能与噩梦有关？）
2. 江城翻阅胡燕病例，是否在调查什么？
3. 胖子晚上回来后会遇到什么？（门还在那里）
4. 江城的"青春期"说法是玩笑还是另有含义？
