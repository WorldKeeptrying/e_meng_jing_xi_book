# 第十七章 - 归途

## 核心事件

### 1. 暖姐尸体位置确认
- 陈晓萌承认：昨晚她到时暖姐尸体在卧室门后
- 警告：如果江城昨晚再往里走几步，鬼会连他的尸体也拖到三层
- 江城问：暖姐尸体**现在**在哪里？

### 2. 三层暗房真面目 - 刑房

#### 灯亮前的发现
- 陈晓萌在墙边摸索到按钮，按下后灯亮
- 房间景象清晰：**教堂式布置**

#### 刑房环境
| 元素 | 描述 | 意义 |
|------|------|------|
| 倒立十字架 | 墙上巨大十字架，倒立 | 魔鬼象征，异端信仰 |
| 暖姐尸体 | 头朝下被绑在倒立十字架上，脸色扭曲 | 确认死亡 |
| 刑具 | 各式各样，形制古朴血腥，中世纪风格，带血迹 | 长期虐待工具 |
| 整体性质 | **刑房** | 别墅主人是异端信徒 |

#### 关键推断
- 刑房布置非短时间能完成 → 刑具拥有者是**别墅主人**，非外来者
- 这家人是**魔鬼信徒**（倒立十字架 = 魔鬼象征）

### 3. 真相揭露 - 一家人才是凶手

#### 胖子恍然大悟
- **真正的凶手是别墅里的一家人**
- 两个男人是**受害者**，非入侵者

#### 客车上的家人对应关系
| 客车上的人 | 真实身份 | 备注 |
|-----------|---------|------|
| 男司机 | 男主人 | 开车的 |
| 带孩子的母亲 | 女主人 | 统筹大局 |
| 女性售票员（郑导） | 女儿1 | 伪装成导游，网上发钓鱼帖 |
| 情侣中的女孩 | 女儿2 | 男性打扮 |
| 情侣中的男人 | **受害者1** | 始终戴眼罩，一动不动 |
| 母子中的大男孩 | **受害者2** | 眼神恐惧，一言不发，手指缠绷带 |

#### 地狱空荡荡，魔鬼在人间
- 女鬼不是为了复仇
- 她只是在**继续生前的恶**
- 刻在骨子里的东西改不掉

### 4. 任务禁忌揭示
- **本次禁忌：伪装**
- 鬼生前通过伪装骗受害者入别墅
- 死后变成鬼延续了这一特性
- 樊力等人因没识破伪装被杀

### 5. 照片线索（陈晓萌提供）
- 原墙上的照片突然清晰（暖姐死后触发）
- 木质相框被拆掉（嫌费事）
- 照片内容：
  - 一家四口合照（能对应客车上的人）
  - 背景就是这间刑具室
  - 两个男人被铁链锁着，穿的衣服与客车上两个受害者一样

### 6. 黑色铁门开启
- 门位置散发淡色幽光
- 空间开始颤抖
- 铁门完全打开，门后是**灰蒙蒙的雾气**
- 陈晓萌第一个迈入铁门消失
- 江城看了眼胖子，也迈入雾气

### 7. 返回现实
- 诡异眩晕感后，踩到坚硬平整的木地板
- 江城回到自己的心理工作室
- 窗外黑夜，月色透入
- 一切与离开时一模一样

### 8. 胖子出现在江城家
- 江城从口袋掏出一张对折多次的纸
- **一只手悄无声息搭上他右肩**
- 江城反应：
  - 慢慢举起双手
  - 主动告知钱财位置（办公桌左侧第三个抽屉暗格，两万块）
  - 钥匙位置（厨房倒数第二个咖啡杯里）
  - 建议劫别的去楼上，"没有床只有床垫，得将就"
- 手抖了抖，中气十足的声音："兄弟，你住的地方不错啊！"
- 江城反手扭住对方，将胖胖身躯压在墙上
- **是胖子**

#### 胖子为何在江城家？
- 樊力说：活着结束任务会回到现实出发点（各回各家）
- 但这里不是胖子家
- 胖子解释：进入门前想着"要是能和兄弟分在一起就好了"
- **江城拆穿**："你是想让我带你过噩梦结界"

## 重要规则

### 返回现实机制
1. 迈入黑色铁门后的雾气
2. 诡异眩晕感
3. 回到现实出发点（但胖子例外，出现在江城家）

### 任务禁忌：伪装
- 鬼生前擅长伪装骗人
- 死后变成鬼延续伪装特性
- 识破伪装是生存关键

### 梦界与现实的联系
- 从梦界带回物品（报纸）
- 可能通过特定意愿改变返回位置（胖子出现在江城家）

## 伏笔与悬念

1. **江城口袋里的报纸从哪来？** - 陈晓萌给的？还是其他人？
2. **胖子为何能出现在江城家？** - 意愿改变了返回位置？
3. **胖子想"带过噩梦结界"** - 下次任务怎么办？
4. **暖姐尸体被绑在倒立十字架上** - 有什么宗教仪式意义？
5. **报纸日期？** - 案件是什么时候发生的？
