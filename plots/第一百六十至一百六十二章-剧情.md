# 第一百六十至一百六十二章 剧情总结

## 第一百六十章 复仇

### 关键事件

1. **阿苏木的抉择**
   - 高壮汉子阿苏木挡在江城身前，表明立场
   - 他用暗门机关封住铁门，为江城等人争取逃跑时间
   - 指点衣柜后的暗门通向隔壁街糕点铺子

2. **陈晓萌与江城逃离**
   - 陈晓萌毫不犹豫冲向铁门，消失在门内
   - 江城进入铁门的瞬间，突然想起：没有发现钱建秀尸体，也没有看到鬼

3. **鬼现身**
   - 江城回头看见衣柜门裂开，血红色眼眸出现
   - 一具血肉模糊的无头尸身倒在房间
   - 扭曲的人影撞破暗门冲向走廊

4. **走廊大屠杀**
   - 村长等人的咒骂声立即消失
   - 尖叫声、撕裂声、鲜血泼洒声响彻走廊
   - 藏在床下的人也被拖出撕成两半
   - 复仇的鬼带着极致恶意降临人间

5. **规则揭示**
   - 鬼在等待他们犯错：打开衣柜门就会触发攻击
   - 如果不主动开门，鬼也无法伤害他们
   - 鬼代表着极致恶意，唯有规则能限制

6. **村长与络腮胡的结局**
   - 鬼当着村长面将络腮胡的头扭了180度
   - 村长面临死亡，欠下的债终究要还

7. **阿苏木挡在村长身前**
   - 一根木棍打断了鬼探向村长的手臂
   - 阿苏木持棍挡在村长身前

## 第一百六十一章 阿苏木

### 关键事件

1. **阿苏木对峙鬼**
   - 2米高壮汉挥舞木棍对抗1米5的鬼
   - 阿苏木虽然害怕，但仍然挺身而出
   - 村长催促他离开，因为阿苏木没有杀人命债

2. **村长与阿苏木诀别**
   - 村长叫他快走，说"长生天记得每一笔债"
   - 阿苏木用后背将村长推向楼梯
   - 随后挥舞木棍冲向鬼

3. **阿苏木之死**
   - 转瞬之间，阿苏木被杀死，头被扭下
   - 村长跌下楼梯，瘫坐在地
   - 阿苏木的头滚落到村长身前，眼睛睁大

4. **阿苏木的身世回忆**
   - 阿苏木来自草原，是逃难过来的
   - 母亲误踩捕兽夹，被老猎人发现
   - 10岁的阿苏木曾徒手勒死一头狼保护母亲
   - 村长动了恻隐之心收留了他们
   - 母亲几年后病逝

5. **阿苏木的成长**
   - 身高近1米9，力大无穷
   - 村长大儿子酒后想教训他，被他单手扔进猪圈
   - 对所有人都善良，只对村长最好
   - 村里人都知道，村长能坐稳位置，阿苏木功不可没

6. **村长的愧疚**
   - 村长曾想带阿苏木一起做"生意"
   - 阿苏木每次都拒绝，劝村长不要再做
   - 村长曾想帮他找女人，但其他村子因小石涧村的名声拒绝
   - 村长视他为第四个儿子，比亲生的还亲

## 第一百六十二章 圈套

### 关键事件

1. **村长之死**
   - 村长精神崩溃，呢喃后倒地
   - 鬼挖出他的心脏，头也没保住
   - 三颗人头（村长、钱建秀、阿苏木）被丢在暗室

2. **江城发现异常**
   - 鬼离开后，江城感到被注视的目光
   - 衣柜门严丝合缝，暗室内没有活人
   - 阿苏木歪斜的头颅直直盯着自己
   - 更诡异的是：阿苏木的头颅转向了另一个方向

3. **精神控制**
   - 江城无法移开视线
   - 阿苏木的眼睛像寒潭，让人无法自拔
   - 右脚不受控制抬起，接近光晕边缘
   - 意识到这是鬼的圈套

4. **挣扎与抗争**
   - 江城想到寒贞潭的经历
   - 鬼没有离开，仍在暗处观察
   - 她无法伤害门内的江城，但可以影响精神
   - 咬舌尖想唤醒自己，但感受不到痛感
   - 阿苏木的声音在耳边回荡，哀求又诱惑

5. **最后时刻**
   - 江城竭力拖延时间，等待铁门消失
   - 眼前突然出现一张支离破碎的脸
   - 左脸完全裂开，细框眼镜挂在耳边
   - 牙齿散落在血肉中——这才是鬼的真面目
   - 鬼气急败坏盯着江城，再迟几秒他就会死

6. **返回现实**
   - 江城被巨大吸力扯回现实
   - 回到自己的办公室
   - 铁门消失，水还是温的
   - 胖子听到动静下楼，惊讶于江城"完事"之快

### 本章要点
- 鬼虽然复仇成功，但并未真正离开
- 她设下圈套，试图杀死江城
- 规则限制鬼不能主动攻击门内的人，但可以影响精神
- 江城凭借意志力撑到铁门消失，成功脱险
- 阿苏木的头是鬼精心设计的陷阱
