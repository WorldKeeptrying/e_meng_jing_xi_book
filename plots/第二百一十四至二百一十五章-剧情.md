# 第二百一十四至二百一十五章 合作/成交

## 章节概述

尘然继续折磨苏小小，揭示其扭曲的"教化"哲学。苏小小在绝境中反将一军，暗示她另有合作伙伴。尘然识破后踩断苏小小脚踝，却意外触发石盘血祭机制。江城和胖子现身，与苏小小达成交易后离开，苏小小以身为饵阻止骸骨复苏，壮烈牺牲。

## 剧情详述

### 第二百一十四章 合作

#### 尘然的"教化"哲学
- 尘然舔着嘴唇逼近苏小小，准备砍下她的四肢
- 他的扭曲理念：
  - **折磨不是目的，教化才是**
  - 喜欢看人临死前拼命求生的模样
  - 让人明白生命的可贵
  - 会将半死不活的人丢入门内"放生"
- 他更习惯砍断四肢：方便搬运，止血只需在开门前很短时间处理
- 自比贝多芬、梵高、尼采：
  - "有些疯子只是不被世人理解"
  - 尼采的"上帝已死"令他痴迷
  - **深红是他的归宿**

#### 苏小小的反击
- 苏小小重伤吐血（肋骨刺穿肺部）
- 被尘然踩断手腕
- 面对即将被砍四肢的命运，她却在笑
- **关键台词**："就算你能杀掉我，你也斗不过他的"
- 暗示：郝帅等人没出现是有原因的
- 尘然脸色阴沉："看来苏小姐的合作伙伴不止我一个"

### 第二百一十五章 成交

#### 石盘血祭触发
- 尘然踩断苏小小脚踝，骨肉粘连声令人作呕
- 苏小小的血浸入土壤，流入石盘
- **诡异事件发生**：
  - 淡红色光晕展开，形成血色屏障
  - 骸骨颤动，仿佛无数甲壳类生物逼近
  - 惨白骸骨上不断聚集血肉
  - 白色石盘变成修罗场

#### 强制画面注入
- 尘然眼前出现诡异画面（不通过眼睛，直接映入脑海）：
  - 巨大木船、肩上武器、铁索
  - 穿着单薄古怪的人围在一起叩拜哭泣
  - 女人将鲜血倾倒身上
  - 头顶乌鸦盘旋，地上海鸟尸体
- 感受到强烈不甘与怨恨
- 尘然顿悟后选择离开

#### 江城与苏小小达成交易
- 江城（医生）快速跑到苏小小身边
- 胖子紧随其后
- 江城判断：**苏小小伤重，无法带走**
- 苏小小早有准备，平静接受

#### 交易内容
- **江城**："你告诉我线索，我尽可能救你弟弟"
- **苏小小**："成交！"
- **关键线索**：
  1. 第一扇石门外的**石柱侧面**有线索，刻有暗纹画
  2. **线索报纸在尘然手上**
- 时间紧迫：石盘上的骸骨已初具模型，即将展开猎杀

#### 苏小小的最后抉择
- 江城将苏小小托付给尘然（原话："那就拜托了"）
- 江城拉起胖子向森林狂奔
- 苏小小挣扎站起，抓住尘然丢下的长矛
- **她的选择**：
  - 扑向石盘
  - 长出的血肉再次剥落
  - 骸骨摔倒，血光逆向旋转
  - **一切恢复原状，仿佛从未有人来过**

## 关键信息

### 尘然的真面目
| 层面 | 内容 |
|------|------|
| 表面 | 温文尔雅的合作者 |
| 实际 | 享受折磨的疯子 |
| 哲学 | 折磨即教化，让人珍惜生命 |
| 偶像 | 尼采——"上帝已死"的狂人 |
| 弱点 | 对诡异事物有一定敬畏 |

### 苏小小牺牲的代价
- **四肢全断**：一手臂+一脚踝被尘然踩碎
- **内脏重伤**：肋骨刺穿肺部
- **以身殉道**：用鲜血触发石盘，再用身体阻止复苏
- **结局**：生死不明（极可能死亡）

### 石盘的秘密
- 需要**鲜血**触发机制
- 会让骸骨**复活并猎杀**
- 苏小小用**自己的血**使其复苏
- 又用**自己的身体**使其归位
- 笼罩空间的血光可逆向旋转
