# 第十五章 - 时间

## 核心事件

### 1. 时间紧迫性
- 白天剩余时间最多3小时，已过去一半
- 黑夜降临后将进入**永夜**
- 江城决定分组搜索：江城+胖子一组（一层），樊力单独一组（二层）

### 2. 绳子线索发现
- 江城在楼梯侧后方角落找到一根绳子
- 绳子一端系在二层走廊拐角（昨晚江城和胖子躲藏的位置）
- **关键异常**：昨晚这里没有绳子 → 绳子是**后来出现**的

### 3. 储存室异常
- 储存木柴的小房间门虚掩着
- 樊力发现异常：昨天他用绳子和钉子系住了门把手，**绳子不见了**
- 胖子在木柴后找到断成两截的绳子，**断口整齐** → 被匕首类工具割断
- 储存室后面的门前木柴被搬空

### 4. 车辙印之谜
- 透过储存室后门玻璃看到泥泞地面上有**客车车辙印**
- 奇怪之处：附近都是森林，**没有客车可以开进来的路**
- 推断暖姐和陈晓萌通过某种方式离开了

### 5. 时间规律揭示（樊力&江城推理）

| 关键时间点 | 事件 |
|-----------|------|
| 3:07 | 谢雨死亡时手表显示的时间 |
| 3:07 | 樊力听到古怪声音时怀表显示的时间 |
| 6:00 | 不久前看怀表显示的时间 |
| **规律** | 所有计时工具在某一时刻会**突然统一到3:07** |

- 胖子恍然大悟：江城在书房等3:07，因为到那时鬼会出现
- 鬼没来 → 江城去试探找暖姐 → 看到"暖姐"活着后说樊力已死

### 6. 客车出现
- 樊力数过：别墅后有**6道车辙印** → 客车来过**3次** → 对应他们在别墅度过的**3个夜晚**
- 天黑后，三人等待特定时间
- 客车准时出现，停在别墅约20米处
- 樊力推开了储存室后门 → **门开了**

### 7. 郑导的客车
- 车上人员：那对情侣、一对母子、导游郑导
- 郑导热情邀请："各位旅客久等了！现在我们要启程回家了。"
- **樊力警觉**：持刀逼退郑导和司机，自己第一个上车
- **江城反常**：没有上车

## 关键线索

| 线索 | 位置 | 推断 |
|------|------|------|
| 绳子 | 二层走廊拐角 | 昨晚没有，今早出现 → 有人布置 |
| 断绳 | 储存室木柴后 | 被匕首割断，断口整齐 |
| 车辙印 | 别墅后泥地 | 6道车辙 = 客车来过3次 |
| 计时异常 | 3:07 vs 6:00 | 特定时间所有计时工具统一到3:07 |
| 无鞋印 | 别墅后泥地 | 只有车辙没有脚印 → 人未离开 |

## 重要规则

### 时间机制
1. 计时工具平时**不准确**
2. 但在**特定时刻**会突然统一到**3:07**
3. 3:07是鬼出现的关键时间点
4. 客车在特定时间才会出现，门才能打开

### 永夜预警
- 白天持续缩短，黑夜持续延长
- 进入永夜后，世界将完全陷入黑暗

## 伏笔与悬念

1. **江城为何不上车？** - 他察觉到什么？
2. **车辙印有但无鞋印** - 暖姐和陈晓萌真的离开了吗？
3. **绳子是谁布置的？** - 目的是什么？
4. **樊力为何第一个上车？** - 他的生存策略
