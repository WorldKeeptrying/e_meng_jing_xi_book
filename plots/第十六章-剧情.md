# 第十六章 - 陷阱

## 核心事件

### 1. 客车陷阱揭露
- 郑导与江城对视时脸色变难看，眼神变**怨毒**
- 车门快速关闭，客车飞快开走
- 胖子后知后觉：**郑导是鬼！这是陷阱！**
- 江城摇头确认：确实不是回去的路

### 2. 三层铁门自动开启
- 江城回到别墅，走向通向三层的铁门
- 铁门**从内部被打开** → 有人在里面
- 出现的是**陈晓萌**，人畜无害的模样

### 3. 三层暗房环境
- 无窗，照明：两根婴儿手臂粗的白蜡烛
- 面积很大，比主卧室还大
- 墙上嵌着**纯黑色铁门** → 离开梦界的关键
- 胖子看到黑色铁门才彻底放心（感觉无法伪造）

### 4. 江城识破骗局的完整推理

#### 第一层推理：自愿暴露
- 江城并非被骗出别墅，而是**自愿的**
- 关键证据：别墅后泥地有**车辙印但没有鞋印**
- 推断：陈晓萌等人**根本没离开别墅**

#### 第二层推理：暖姐已死
- 疑问：如果她们离开了，为什么假暖姐还会出现在卧室？
- 证据：**玳瑁材质方形纽扣**（在主卧室找到）
  - 带着扯断的线头
  - 是暖姐大衣上的
- 还原过程：
  1. 鬼杀死暖姐时纽扣在挣扎中飞出
  2. 陈晓萌等鬼离开后收拾了房间
  3. 目的：防止发现暖姐被杀，营造已逃离的假象

#### 第三层推理：心理压制
- 江城用嘴炮攻击陈晓萌："其貌不扬五短身材自以为是又呆又笨还平胸的蠢女人"
- 心理战术：
  - 强调密室环境，孤男寡女
  - 暗示可能发生"不友好的事情"
  - 故意加上胖子："这里还有一个胖子，你考虑清楚"
- 效果：陈晓萌脸色变白，被迫配合

### 5. 问答环节

**江城问**：这扇门你是怎么打开的？

**陈晓萌答**：昨天你走后，杀了暖姐的那只鬼将暖姐尸体拖到这里，**我跟着它进来的**

### 6. 关键信息

| 事件 | 时间 | 详情 |
|------|------|------|
| 暖姐死亡 | 昨晚江城离开后 | 鬼杀死并拖到三层 |
| 陈晓萌行动 | 跟随鬼进入三层 | 知道了开门方法 |
| 布置骗局 | 今早 | 绳子、断绳、车辙印 |
| 距离铁门 | 5米 | 陈晓萌距离铁门 |
| 江城距离 | 3米 | 江城距离陈晓萌 |

## 人物关系变化

### 江城 vs 陈晓萌
- 从合作者变为**对立者**
- 江城完全压制陈晓萌，掌握主动权
- 陈晓萌被激怒但被迫配合

### 胖子角色
- 再次成为旁观者/气氛调节者
- "这里不止你一个孤男，还有我"
- 被江城利用作为心理压制工具

## 关键线索

| 线索 | 意义 |
|------|------|
| 玳瑁纽扣 | 证明暖姐死于挣扎，非自然离开 |
| 无鞋印 | 证明陈晓萌未离开别墅 |
| 黑色铁门 | 离开梦界的出口，需特定方法打开 |
| 鬼拖尸体 | 鬼会带尸体到三层 → 三层是特殊区域 |

## 重要规则补充

### 黑色铁门开启机制
1. 需要找到所有线索
2. 在**特定时间**才会出现/开启
3. 可以跟随鬼进入
4. 陈晓萌通过跟踪鬼学会了开门方法

### 鬼的行为模式
1. 会伪装成人（郑导、假暖姐）
2. 会设置陷阱（客车骗局）
3. 会带尸体到三层（原因不明）
4. 有怨毒眼神（被识破后）

## 伏笔与悬念

1. **陈晓萌为何能跟随鬼？** - 她有什么特殊能力？
2. **暖姐尸体在哪里？** - 三层暗房中？
3. **黑色铁门后是什么？** - 真正的出口？
4. **鬼为什么拖尸体到三层？** - 有什么仪式或目的？
5. **距离开门还有不到10分钟** - 会发生什么？
