# 第一百一十六至一百一十八章 剧情总结

## 第一百一十六章 白烛

### 早晨冲突
- **奔富反应**：听到江城与陈晓萌对话后鼻血不止，手忙脚乱
- **陈晓萌恼羞成怒**：拿出绳索要勒死江城
- **众人反应**：盯着绳索产生奇怪想法（证实某些猜想）
- **早餐过程**：
  - 陈晓萌气得只喝几口粥
  - 江城喝了三碗，吃光腌黄瓜
  - 江城试探问陈晓萌是否还喝，看到铁青脸色作罢

### 村长到来
- 站在旅社门前不敢进入（因昨天与店老板不愉快）
- 不时向里张望
- **周荣交涉**：表示要先还钥匙再出发
- **村长态度**：兴奋搓手，"你们肯来就是我们全村人祖上积德"
- **于蔓讽刺**："要是祖上积了德，你们也不至于落到这副下场"（小声）

### 旅社异常继续
- **店老板夫妇失踪**：从早上到现在未出现
- **早餐准备**：粥和8副碗筷一早放在楼下
- **江城与陈晓萌警觉**：都盯着楼梯等待店老板出现

### 周荣与江城上楼调查
1. **钥匙归还任务**：周荣上楼寻找店老板
2. **江城陪同**："我陪你一起去吧"
3. **周荣印象**：因昨天支持而对江城印象较好

### 三层调查发现
1. **寻找无果**：二层三层都没有店老板夫妇
2. **旅社客人**：只住着他们8位客人
3. **江城调侃**：称赞周荣将钥匙放在柜台上的办法
4. **江城奇怪比喻**："在下与你相比，就如曜日与此烛光"

### 三层诡异细节
1. **结构异常**：
   - 三层10间房（一面5间，两两相对）
   - 一侧楼梯，另一侧被墙堵死
   - 墙面漆成黑色，采光极差

2. **蜡烛诡异**：
   - 惨白色蜡烛幽幽燃烧
   - 烛台位于墙正中央，照明作用有限
   - 像灵堂布置，给人压抑感
   - **火苗颤抖**：突然猛烈颤抖，有微风拂过

3. **周荣推理**：
   - 外部看二层三层应一样长，实际三层短
   - 推测**黑色墙后还有空间**
   - 店老板夫妇可能藏在暗室中

### 暗室确认
- **发现缝隙**：在黑色墙体发现粗糙缝隙
- **偷窥所见**：
  - 暗室内有同样白色蜡烛
  - 店老板背影，围着脏围裙
  - 店老板激动解释着什么
  - 看到一双小脚，不是老板娘的

## 第一百一十七章 高人

### 暗室真相揭示
1. **周荣看到的恐怖景象**：
   - **老女人**：坐在木椅上
   - **恐怖面容**：半张脸被融化，只剩下一只眼睛，鼻子、嘴唇都没有
   - **店老板**：面色狰狞扭曲，手握尖刀

2. **周荣惊吓反应**：
   - 受极大惊吓猛地向后缩
   - 碰倒蜡烛，被江城迅速接住
   - 两人躲入308房间

### 店老板的异常行为
1. **寻找入侵者**：
   - 听到动静后立即寻找
   - 手握**闪着寒芒的尖刀**
   - 面色狰狞扭曲，与昨日老好人形象判若两人

2. **警惕性高**：
   - 使用备用钥匙开门搜索
   - 第一次搜索无果后假装离开
   - **杀回马枪**：三分钟后返回再次检查
   - **第二次回马枪**：5分钟后再次检查

### 周荣与江城秘密交流
1. **周荣的合作提议**：
   - "我想与你联手，尹先生"
   - 透露已通过**5次任务**，这是第6次
   - 认为这次任务难度极高，**前5次加起来都不如**
   - 还没进入"域"就已发生异常事件

2. **重要信息透露**：
   - **"域"的概念**：鬼可以行动或干扰到的范围
   - 判断**安平镇不在"域"内**（鬼不能到达或干扰的范围）

3. **江城的应对策略**：
   - 询问"域"概念，让周荣产生怀疑（瞬间疑惑→怀疑→恢复）
   - **不深究**：知道是自己认知盲区
   - **保持高深莫测**：既不答应也不拒绝合作提议
   - **眼神暗示**：让周荣自己思考

### 返回楼下后的应对
1. **隐瞒真相**：对暗室事件只字不提
2. **陈晓萌怀疑**：
   - 质问两人在楼上长时间做什么
   - 试图泼脏水，怀疑两人关系
   - 不惜波及周荣

3. **江城反击**：
   - "龙儿，你在说什么？昨晚我有多勉强...你又不是不知道"
   - 陈晓萌眼前一黑，差点呕血

4. **奔富反应**：看向江城眼神愈发炽热

### 出发前往小石涧村
- **村长催促**：天色不早，山路崎岖不好走
- **江城同意**："我们这就出发吧"
- **村民反应**：脸上乐开了花

## 第一百一十八章 入村

### 前往小石涧村途中
1. **裴乾与村长交流**：
   - 谈论当地风土人情
   - 江城眯着眼睛听
   - 得知村子处于**两座山的夹缝中**

2. **江城行为**：
   - 想勒死江城，但江城看穿她的心思
   - 上厕所都要拉上一个人
   - **最近一次拽于蔓**：两人隔一刻钟才回来

3. **江城特权要求**：
   - 看到裴乾有人打伞，嚷嚷"我也热，我也要壮汉给我打伞"
   - **自称高人**："谁是你兄弟？我是高人！"
   - 村长无奈，让黝黑汉子给江城打伞

4. **江城与于蔓互动**：
   - 热情邀请于蔓来伞下并肩而行
   - 两人职业类似，有说不完的话题
   - 江城小声说话惹于蔓俏脸泛红
   - 两人玩味打量陈晓萌，眉目交流

### 到达小石涧村
1. **路程艰难**：
   - 停停走走，天快黑才望见村落
   - 夕阳笼罩下的山村别有一番韵味

2. **村落景致**：
   - 白房黑瓦间层次分明
   - 屋顶炊烟袅袅升起
   - 裴乾赞赏：出自真心，如晕染开的中国画

3. **进入村落**：
   - 蹚过清澈溪水（堪堪没过脚踝）
   - 青石板路蜿蜒没入村落
   - 村落约百十户人家

### 住宿安排困难
1. **第一间院落拒绝**：
   - 几间低矮瓦房用栅栏围成院子
   - 江城注意到簸箕中新鲜地瓜（带湿润泥土）
   - **争执发生**：村长阴沉着脸灰溜溜出来
   - **女人冷漠**：手拎烧黑木棍，神色冷漠

2. **借口变化**：
   - 第一家：男人不在，不方便
   - 第二家：同样拒绝，说辞换了
   - 最终安排：偏僻破败院落

### 最终住宿地点
1. **院落描述**：
   - 三间房：西侧阔气大房（大门紧闭，挂黑色铜插锁）
   - 东侧两间小房
   - 木门老旧，门轴发出刺耳噪音

2. **房间内部**：
   - 二十几平米小厅
   - 中间有不知材料垒成的炉子
   - 炉内有早已熄灭的煤渣

3. **村长熟悉**：
   - 轻车熟路翻出木炭
   - 黝黑汉子抱柴火进来
   - 炉火噼里啪啦烧起来

### 住宿安排
- **村长建议**：一间房住4人，两人住一侧，两人住另一侧
- **裴乾拒绝**："我们都住在这间房就可以了"
- **村长改口**：从"随高人你的"改为"随高人你们的意思"
- **江城眼神**：哼哼唧唧终究没说什么
- **村长紧张**：舔嘴唇，仿佛逃过一劫

### 村长讲述恐怖故事开端
1. **于蔓催促**："这么远请我们来不会是游山玩水吧"
2. **村长准备**：
   - 关闭房门，用木棍插上充作门栓
   - 脸色变得复杂：恐惧、无奈、懊悔杂糅

3. **恐怖开场**：
   - "你们...相信有鬼吗？"
   - **周荣敷衍**："相信，举头三尺有神明"
   - **村长强调**："是那种十分恐怖的，能杀人的厉鬼"

4. **诡异现象**：
   - 所有人觉察到一股寒意
   - 寒意来得突然且诡异
   - **炉火受影响**：在注视下黯淡了许多
