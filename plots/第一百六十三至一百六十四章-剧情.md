# 第一百六十三至一百六十四章 剧情总结

## 第一百六十三章 源头

### 关键事件

1. **时间流速差异**
   - 江城在噩梦世界待了几天，现实世界仅过去几分钟
   - 胖子从卧室算起约20分钟，但实际噩梦时间更短
   - 铁门是连接两个世界的通道

2. **胖子的预感**
   - 江城即将出来时，胖子脑海中突然有电流闪过
   - 产生奇怪的感觉，预感出事
   - 这种预感让他意识到江城回来了

3. **灵异电台的故事**
   - 胖子深夜开大车时听到的灵异节目
   - 讨论各种见鬼方式：
     - 请笔仙：朋友第二天摔断腿
     - 十字街头敲夹生米碗：看到十几个饿死鬼黑影
     - 室内打伞、弯腰从胯下看、午夜对镜念名字等

4. **脑电波理论**
   - 最后一位连线者介绍的方式：脑波/脑电波
   - 核心观点：见到鬼需要脑电波与鬼一致
   - "你在想着见到它时，鬼同时也在想着你"
   - 节目现场嘉宾和主持人因此噤声

5. **干扰现象**
   - 胖子后来才意识到：听不清那人说话不是因为口齿不清
   - 而是有东西在那人身边，脑电波频率接近产生干扰
   - 类似电磁波干扰原理

6. **关联线索**
   - 江城想起胡燕录音中也有大段杂音
   - 暗示可能存在类似现象

## 第一百六十四章 复盘

### 关键事件

1. **日常场景**
   - 凌晨胖子守在卧室门外，将沙发挪到门前
   - 江城看到门外有颗大脑袋盯着自己
   - 清晨江城在楼下挪收纳架，胖子帮忙

2. **胖子的照顾**
   - 胖子为江城准备早餐：麦片+牛奶+煮鸡蛋
   - 洗了三遍手确保没鱼腥味
   - 问中午吃糖醋鱼还是水煮鱼
   - 江城选择糖醋鱼（省调料）

3. **任务复盘开始**
   - 江城告诉胖子：没拿到任务奖励
   - 有人抢先一步从门离开（李璐）
   - 遇到了熟人陈晓萌

4. **任务细节回顾**
   - 江城讲述挖出地雷换位置埋到村口
   - 讲述阿苏木的故事
   - 阿苏木虽不算好人，但最后抉择令人钦佩
   - 善恶不过一念之间

5. **李璐的阴谋**
   - 李璐太厉害，瞒过了江城和陈晓萌两人
   - 设圈套想杀他们
   - 利用了两人相互防备的心理

6. **胖子的好奇**
   - 胖子问江城和陈晓萌住在一起的晚上发生了什么
   - 满脸好奇和激动

### 本章要点
- 时间规则：噩梦与现实时间流速不同
- 脑电波理论可能是噩梦/见鬼的核心机制
- 江城没有完成任务奖励（被李璐抢先）
- 日常互动展现江城与胖子的默契关系
- 胖子调节能力很强，一觉醒来就恢复
