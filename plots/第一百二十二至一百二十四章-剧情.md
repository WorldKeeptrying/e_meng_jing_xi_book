# 第一百二十二至一百二十四章 剧情总结

## 第一百二十二章 跟踪

### 奔富守夜

1. **守夜安排**
   - 奔富负责第二班守夜
   - 于蔓是第一班
   - 每人守夜两小时，轮换制

2. **意外发现**
   - 通过窗户大洞看到门外有两道人影
   - 认出是**周荣**和**尹志平（江城）**
   - 两人鬼鬼祟祟，背对大门，身形佝偻

3. **联想推测**
   - 联想到清晨两人借口还钥匙，在楼上磨蹭10多分钟
   - 推测两人找到了什么线索
   - 决定跟踪一探究竟

### 跟踪行动

1. **路线选择**
   - 周荣二人选择难走的路，尽量俯低身体融入夜色
   - 没有离开院子，而是绕到**废弃建筑**旁

2. **危险行为**
   - 周荣与尹志平翻窗进入建筑
   - 奔富犹豫后决定跟上去
   - 佩服两人脚步声几不可闻

3. **内心独白**
   - 意识到自己太弱，队友都比他强
   - 担心这次任务难度高，能活下来的人寥寥无几
   - 决定另辟蹊径，靠线索活下去
   - 认为杀掉队友会引发可怕后果，所以即便被发现也不怕

### 进入建筑

1. **内部环境**
   - 内部比外面房间豪华得多
   - 十分黑暗，只能看清周围一点点

2. **撞到障碍**
   - 腿撞到什么东西，发出声响
   - 楼上没有任何声音，仿佛两人凭空消失

3. **上楼遭遇**
   - 跑上楼质问周荣和尹志平
   - 两人冷冷看着他，双臂下垂，一言不发
   - 缓缓举起右手，以诡异方式齐齐指向奔富
   - 脖子向右狠狠一歪
   - **奔富转身就跑**

## 第一百二十三章 可怕

### 楼道遇险

1. **判断局势**
   - 意识到眼前的周荣尹志平出了问题
   - 可能不是鬼假扮，就是被鬼操纵的尸体
   - 原先的两人可能已经死了

2. **黑影出现**
   - 转身没跑几步，楼梯上多出一团黑影
   - 乌云散开，月光照入
   - 看清那不是黑影，而是一个**蹲在地上的人**

### 众人反应

1. **发现奔富离开**
   - 蒋忠义守夜时发现鬼鬼祟祟的影子
   - 隐约记得是奔富的后背
   - 黑影一闪消失在视野中

2. **江城反应异常**
   - 脸色不自然，纠结、遗憾、懊悔
   - 周荣询问时，江城回答："**可惜那个地瓜了**"

3. **陈晓萌分析**
   - 真奔富要么发现线索，要么被什么东西勾引
   - 另一种可能是鬼假扮奔富引他们出去

### 众人外出

1. **江城开门**
   - 老旧门轴发出噪音
   - 江城拉开门走出去
   - 陈晓萌、周荣紧随其后
   - 另一间房的裴乾、于蔓、李璐也出来

2. **李璐证词**
   - 睡觉特别轻，听到风声起来查看
   - 发现门开着，奔富不见了

3. **裴乾判断**
   - 奔富被鬼迷惑，凶多吉少
   - 没人提议去找人

### 守夜调整

1. **安全措施**
   - 所有人聚在一个房间
   - 关紧门，烧旺炉火
   - 守夜人数提升到两人
   - 一半小时一换班

2. **心态变化**
   - 老玩家无人提蠢问题
   - 单纯的兔死狐悲
   - 重点是从中推断任务禁忌

## 第一百二十四章 失踪

### 早餐时分

1. **村长送餐**
   - 村长和黝黑汉子挑着扁担来送食物
   - 一整屉淡黄色杂粮馒头
   - 黑坛子里有水瓢可喝水

2. **陈晓萌观察**
   - 发现村长有意无意查人数
   - 抻着脖子朝房间内瞥
   - 意识到村长早就发现少了一人

### 奔富死讯

1. **村民报信**
   - "镇子上又出事了"
   - 尸体被吊在树上，肚子撕烂，心肺肠子丢在地上

2. **确认身份**
   - 问过村里，没人失踪
   - 确认死者是奔富

3. **村长反应**
   - 欲言又止，不知如何交代
   - 裴乾为其解围："来之前都考虑过风险"

4. **赔偿承诺**
   - 报酬加倍
   - 多出的作为给奔富家人的补偿

### 江城的行为

1. **收集馒头**
   - 将剩下的馒头都塞进衣服里
   - 胸前鼓鼓囊囊，比于蔓还壮观

2. **收买汉子**
   - 掏出一个馒头塞进黝黑汉子袖口
   - 汉子憨厚点头

3. **提关键问题**
   - "为什么奔富在村里失踪，尸体却出现在镇子上？"
   - "所有失踪的人，最后尸体都会出现在镇子上？"

### 关键信息揭露

1. **村长确认**
   - 村子里失踪的人最后尸体都会出现在安平镇周围
   - 推测原因：赵香妹想让所有人知道当年村子做过什么

2. **警察态度**
   - 镇子上警察清楚村里的怪事
   - 睁一只眼闭一只眼，任由村子自己折腾
   - 不方便说出口

3. **监狱消失**
   - 当年买了女人的汉子都被抓起来判了刑
   - 但最后一个个都在监狱中**消失了**

### 前往后山

1. **地理发现**
   - 所住院落在村子正中心
   - 青石板路可将村子一分为二
   - 住的这一侧完全荒废，家家大门紧闭

2. **村落惨状**
   - 门轴烂掉，半扇门斜挂在门框上
   - 离开的人要么被害，要么逃走

3. **最终目标**
   - 前往后山深潭
   - 那里有赵香妹留下的石碑诅咒

## 核心线索与悬念

### 废弃建筑探索

1. **周荣江城行动**：深夜翻窗进入废弃建筑
2. **奔富跟踪**：发现两人上楼后诡异反应
3. **死亡暗示**：两人脖子歪向一侧，举手指向黑影
4. **真实情况**：周荣江城实际在门后观察

### 鬼的行事规律

1. **失踪→死亡**：村子里失踪的人最终都死亡
2. **尸体转移**：尸体出现在安平镇而非村子
3. **惨烈死状**：奔富被吊树上，肚子撕烂
4. **时间线**：夜间失踪，清晨在镇子发现尸体

### 监狱消失事件

1. **当年情况**：买了女人的汉子被警察抓走判刑
2. **诡异结局**：这些人在监狱中一个个消失
3. **暗示**：赵香妹的诅咒范围极广

### 村落荒废真相

1. **地理布局**：院落位于村子正中心
2. **人口流失**：一侧完全荒废，居民或死或逃
3. **村长心态**：请来驱魔人孤注一掷

### 当前谜团

1. **废弃建筑内**：周荣江城究竟发现了什么？
2. **鬼的目标**：为何要转移尸体到镇子？
3. **监狱事件**：当年那些人如何消失的？
4. **石碑诅咒**：亲眼目睹是否会触发诅咒？
5. **江城异常**：为何可惜地瓜？他知道什么？
