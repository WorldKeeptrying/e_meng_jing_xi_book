# 第二十八章 花样 - 剧情总结

## 核心事件

### 1. 404房间内部

**房间状况**：
- **十分干净**（与外面破败形成对比）
- 收拾整洁的床铺
- 擦得干干净净的桌子
- 唯一窗台上摆着一盆叫不出名字的花
- **花呈淡紫色**，透露出一股莫名哀伤的情调
- 与外面年久失修的状态完全不同

**胖子感慨**：
- "被卷入噩梦中的我们，像不像身处乱世中的一朵浮萍"
- "飘摇不定，不知在哪里扎根"
- "过着有一天没一天的日子，或许下一秒死亡就会降临"

**江城回应**：
- 叹了口气："或许吧"
- "但至少有你在身边陪着我"
- 胖子感动，眼泪汪汪
- "没想到我们才认识几天，感情已经升华到了如此地步"

**江城真相**：
- 看着胖子诚挚的眼神，动情道
- "胖子，你好好想一想，如果你都能活着，那么当时所面临的危险能要我的命吗？"
- "我一定是会死在你后面的"
- "所以你放心，如果真有那一天，我会尽力把你的骨灰带出去，埋在我办公室的花盆里"
- "你体格大，骨灰里钙和磷的含量也会比普通人高"

**胖子反应**：
- "医生打小就没人教你怎么说话吧"

### 2. 房间检查

**江城检查结果**：
- 认真检查了一遍房间
- 发现**一个很小很小的柜子上了锁**
- 怎么也打不开
- 暂时还没发现其他可疑的地方

### 3. 饥饿与巧克力

**胖子饥饿**：
- 一头栽倒在床上
- 肚子咕噜咕噜叫
- "医生，这里好奇怪啊，我才刚来，就饿得不行"
- 天已经完全黑了
- 在没有任何线索的情况下黑夜活动愚蠢且危险
- 领路女人特意提醒不要在夜里活动（原因未知）

**江城掏出巧克力**：
- 一大块巧克力
- **红色包装**
- 胖子记得在**很小很小的时候流行**的那种
- 胖子瞬间从床上坐起来，眼睛直放光

**巧克力来源**：
- 江城掰了一块丢过去
- 胖子问："你从哪里弄来的？"
- 江城："来的路上"
- 胖子走出咖啡馆时看到附近有不少或大或小的超市
- 但江城身上没有钱（进入噩死后大部分杂物会被剔除，钱也包括在内）
- 江城小口咬巧克力，看黑洞洞窗外，慢慢咀嚼
- "不是买的"

**胖子猜测**：
- "那是怎么来的？"
- 下一瞬间蓦然瞪大眼睛
- 惊讶道："你不会是偷的吧？"

**江城解释**：
- 不满望向胖子："胡说！"
- "是超市的老板娘硬要送给我的，我说不要她还生气了"

**胖子质疑**：
- 狐疑盯着江城
- "这么贱的老板娘哪里有，请给我来一打"
- 江城放下巧克力要和胖子大战三百回合
- 胖子主动放弃，举手道
- "医生，我现在都被你带的跑偏了，咱们现在可是在噩梦里，可能明天就会死"

**江城安慰**：
- 拍他肩膀
- "不要那么悲观"
- "你可能连今天晚上都活不过去"
- 胖子："……"

### 4. 407房间状况

**恐龙睡衣女**：
- 蜷缩在床上
- 用被子蒙着大半个身子
- 不停地哆嗦
- "姐姐，你觉不觉得这里很冷"
- "我穿着睡衣都冷，你不冷吗？"

**旗袍女**：
- 坐在椅子上
- 进房间第一件事：将整个房间仔仔细细检查了一遍
- 发现：桌子里找到一把**断了一半的梳子**
- 垃圾桶十分干净，上面套着崭新垃圾袋
- 没发现其他异常
- 声线清冷："如果你只是想没事找些话题，让我陪你说话，以此来缓解你的恐惧的话，我劝你就不要白费心思了"
- "那没意义，只会徒劳的分散我们的注意力"
- 恐龙女被气势惊到，不再说话
- 肚子不争气地"咕咕"叫

**旗袍女命令**：
- "转身，把窗帘拉上"
- 恐龙女床挨着窗户边
- 窗户上挂着蓝色窗帘，不算大，但够遮住窗户
- 睡衣女哆哆嗦嗦从被子里伸出手，飞快拉上窗帘

**旗袍女警告**：
- "动作轻点"
- "要是把窗帘扯下来还不知道会惹出什么麻烦"
- 睡衣女愣："会有什么麻烦？"
- 旗袍女看了她几眼，移开视线，没搭理
- 站起身走到自己床铺位置，躺下
- 被子盖到下巴处

### 5. 江城打地铺

**胖子疑问**：
- "有床不睡，你为什么要打地铺？"
- 半晌后灵光乍现，嘴角一颤
- "你觉得这个本里还会有鬼从床底下爬出来，所以提前做好准备？"

**江城解释**：
- "并不是"
- 铺开被子，摆好枕头
- 用力拍几下，直到出现人头大小凹陷
- 舒舒服服躺下去，扭动身体
- "我就是习惯了"

**胖子回忆**：
- 没进过江城现实世界卧室
- 但记得江城说过：要是想劫些除了钱财之外的东西，可以去他的卧室
- 但是没有床，只有个床垫

### 6. 胖子询问针对西装男原因

**胖子提问**：
- 抓抓头发，有些奇怪
- "医生，你为什么要针对那个西服男人？"
- "人家又没得罪咱们"
- "我觉得你这么做完全没有意义，还平白无故的为自己树立了一个不简单的敌人"
- "直白点说，如果你不针对他，我们恐怕也不至于住到这间404房间"

**江城回答**：
- 望着天花板
- 两只手插在自己头下
- 半晌后徐徐说道
- "**是他先耍的花样**"

**胖子反应**：
- 闻言惊了一下
- 坐起身瞪大眼睛
- "你说什么医生？"
- （本章结束，悬念留到下一章）

## 重要线索

### 房间异常
1. **404异常干净**：与外面破败的宿舍楼形成强烈对比
2. **淡紫色花**：窗台上叫不出名字的花，透露哀伤情调
3. **小柜子锁着**：很小很小的柜子，怎么也打不开
4. **407有断梳子**：桌子里找到断了一半的梳子
5. **垃圾桶干净**：套着崭新垃圾袋

### 巧克力来源之谜
- 红色包装，很小很小的时候流行的那种
- 进入噩死后身上杂物会被剔除（包括钱）
- 江城说"不是买的"
- 说是"超市老板娘硬要送的"
- 胖子质疑，江城没进一步解释
- **暗示**：江城可能在进入噩梦过程中有什么特殊遭遇/能力

### 江城打地铺
- 不是防备床底有鬼
- 只是习惯了（现实中也睡床垫）
- 与之前噩梦中睡地板保持一致

### 饥饿感
- 胖子刚来就饿得不行
- 梦里饥饿感是否正常？
- 与之前噩梦中饥饿信号一致（胖子肚子咕噜叫）

### 温度异常
- 恐龙女觉得冷（穿着睡衣都冷）
- 旗袍女没觉得冷
- 是心理作用还是真实温度变化？

- 胖子感动又被噎
- 胖子询问针对西装男原因

1. **巧克力来源**：江城到底怎么得到的？老板娘是谁？
2. **小柜子**：锁着的是什么？为什么打不开？
