# 第五十三至五十五章-剧情

## 第五十三章 规定

### 余雯小组的调查行动

**前往彩排女人住处**
- **路线**：校园南侧，经过废弃网球场，到达一栋与宿舍楼相似的建筑
- **环境特点**：
  1. 荒芜区域，晨练女生目光奇怪地盯着周太福
  2. 废弃网球场：围栏锈迹斑斑，地面陈旧，球网消失
  3. 目标建筑：有人居住，窗户有花卉、衣服、脸盆等生活用品

**与彩排女人（冯老师）的对话**
- **身份确认**：冯老师，负责彩排的年轻女教师
- **开场**：女人对余雯等人的到来感到惊讶和犹豫
- **直接切入**：余雯直接进入房间，女人被迫接受

**学校的神秘规定**
- **核心规定**：天黑后尽量不要在校园里闲逛，尤其是不要去教学C座附近
- **教师态度**：老教师对此闭口不谈，只说是学校规定必须遵守
- **冯老师的违规**：第一年工作时，天黑后返回C座办公室取钥匙

### 冯老师的恐怖经历（开始）

**雨夜返回C座**
- **时间**：冯老师刚来学校的第一年，雨夜
- **发现**：C座大门用两条粗铁链锁着
- **意外**：推门时发现锁是开着的，只是摆个样子
- **办公室位置**：C座3层，304房间

**C座的诡异氛围**
- **黑暗特点**：伸手不见五指，仿佛在吞噬黑夜
- **冯老师的感受**："那栋楼就像是活的！它是有生命的，它在等着我自投罗网！"
- **余雯的冷静**：安抚冯老师，强调现在是白天，她很安全

## 第五十四章 雨夜

### 冯老师的恐怖经历（继续）

**进入C座后的异常**
- **环境**：整栋楼非常安静，只有几盏应急灯幽幽亮着
- **冯老师的行动**：打开手机电筒，一口气跑到三楼，进入办公室后立即反锁门
- **直觉**：不知道为什么，就是觉得不对劲

**诡异的脚步声**
- **发现异常**：脚步声不对，不是一个人的脚步声
- **关键细节**：冯老师穿的是运动鞋，但脚步声中有高跟鞋的声音
- **恐惧反应**：躲在门后一动不动，听了10分钟，门外安静

**办公室内的恐惧**
- **开灯**：将所有灯都打开，房间亮堂堂
- **选择**：不敢留在办公室过夜，决定拿到钥匙后离开
- **意外**：准备开门时，办公室电话突然响起

**保卫科的电话**
- **来电者**：学校保卫科，看到C座办公室亮灯，打电话询问情况
- **冯老师的反应**：快哭出来，第一次觉得有人打电话这么幸福
- **安排**：保卫科答应过来接她，让她在办公室等待

**"保卫人员"的到来**
- **人数**：3-4个人，急匆匆的脚步声
- **领队**：光头，膀大腰圆，十分壮实
- **成员**：两个男人，一个女人，都穿着保安制服
- **态度**：批评冯老师不遵守学校规定
- **领队的异常**：时不时瞥一眼漆黑的走廊，仿佛在警惕着什么

**关键疑点的发现**
- **冯老师的观察**：转身拿雨伞时，突然意识到
- **疑点1**：他们没带伞，也没穿雨衣，衣服像新的一样，完全没有淋湿的痕迹
- **疑点2**：那个女人脚上穿着一双红色高跟鞋

## 第五十五章 高跟鞋

### 冯老师的恐怖经历（高潮）

**进一步的怀疑**
- **身份疑点**：冯老师从未在保卫处见过这几个人
- **规定违反**：保卫处值班时间不允许穿高跟鞋
- **冯老师的应对**：故意磨蹭，找借口拖延时间

**"保卫人员"的异常行为**
- **不进入办公室**：始终站在门外等待，不进去
- **态度变化**：光头男人态度越来越差，脸色阴沉
- **冯老师的策略**：一口咬定东西没找到，绝不离开

**系主任办公室的镜子**
- **布局**：办公室内有单独隔出的系主任办公室
- **镜子**：门上有一大面镜子，为老师整理着装用
- **冯老师的计划**：借口去系主任办公室拿资料，实际想用电话求救

**恐怖的发现**
- **镜子角度**：拉开门时，镜子刚好能照到门口位置
- **冯老师下意识一瞥**：看到门前站着四个人
- **恐怖真相**：他们都没有下巴

### 江城的真相揭露

**早餐对话**
- **胖子的问题**：你怎么知道405那两个家伙会将血手印悄悄擦掉？
- **江城的解释**：任务中被鬼盯上的人势必会被其他人抛弃，这次任务难度不低，甄建仁只靠自己玩不转

**血手印的真相**
- **时间**：凌晨4-5点，窗外一片漆黑
- **江城的行动**：叫醒胖子，准备出去一趟
- **胖子的恐惧**：被江城吓唬说床底下有人影
- **最终决定**：胖子害怕，决定跟江城一起去

**制造血手印**
- **地点**：405房门前
- **工具**：白色小药瓶，装的是血
- **过程**：
  1. 江城将血倒在手掌抹匀
  2. 在405房门上印下血手印
  3. 用剩下的血将每根手指延长到正常的两倍
- **目的**：制造恐怖假象，让甄建仁和罗一以为被鬼盯上

## 关键线索分析

### 冯老师的经历线索
1. **C座的恐怖**：教学楼有生命，会吞噬黑夜
2. **高跟鞋声音**：与旗袍女的死状有关联
3. **"保卫人员"疑点**：
   - 没淋湿的衣服
   - 红色高跟鞋
   - 不进入办公室
   - 没有下巴
4. **学校规定**：天黑后禁止去C座，说明C座是危险核心

### 江城的计谋
1. **血手印真相**：是江城制造的假象
2. **目的**：让甄建仁和罗一产生恐惧，迫使他们更积极调查
3. **心理战术**：利用任务中被鬼盯上的人会被抛弃的规则
4. **时机选择**：凌晨4-5点，确保不会被发现

### 任务关联
1. **旗袍女的死**：下巴被撕掉，与"保卫人员"没有下巴对应
2. **高跟鞋**：红色高跟鞋与脚步声中的高跟鞋声对应
3. **C座**：所有恐怖事件的核心地点
4. **学校秘密**：学校在掩盖C座发生的恐怖事件
