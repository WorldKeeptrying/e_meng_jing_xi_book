# 第一百六十七至一百六十八章 剧情总结

## 第一百六十七章 帮忙

### 关键事件

1. **皮阮来访**
   - 皮阮穿着一身白袍，戴着超级大墨镜
   - 摘下墨镜后脸上伤痕累累：右脸多处抓伤，左眼乌青肿胀
   - 胖子震惊，皮阮看起来很有身份却被打成这副模样
   - 江城似乎对这种情况轻车熟路

2. **皮阮被打原因**
   - 女朋友名下的资产纠纷
   - 女朋友的两个孙女和不成器的孙子约他到酒店"和解"
   - 对方玩阴的，想毁掉他的脸来让女朋友疏远他
   - 资产价值：**6个亿**

3. **协议内容**
   - 如果女朋友不幸或瘫痪，资产三七开
   - 皮阮只想拿三成（1千8百万）
   - 皮阮以为应该拿七成（4.2亿）
   - 江城提醒过他不要贪心

4. **江城答应帮忙**
   - 江城提出两个点：6亿不该叫"一点资产"；提醒过不要贪心
   - 皮阮乖乖听话只拿三成
   - 江城耳语了什么让皮阮包办，皮阮狠狠眨眼答应
   - 江城同意帮忙应对，说自己"心地善良"

5. **请皮阮帮忙找人**
   - 江城拿出牛皮纸袋，里面是几张素描画
   - 四张画：两个女人
   - 前三张是同一女人不同穿着+戴眼镜的样子
   - 第四张是另一个女人
   - 江城的画工精湛，寥寥几笔便有9分相似

## 第一百六十八章 厉害

### 关键事件

1. **江城对陈晓萌的侧写**
   - **年龄**：18至23岁之间
   - **身份**：可能是大学在读，理科
   - **成绩**：比较出色但不拔尖
   - **校园表现**：不会有夸张荣誉，不住公共宿舍
   - **行为特征**：频繁请假早退，与老师同学关系一般，习惯独来独往
   - **关键**：不姓陈

2. **江城对第二个女人的侧写**
   - **年龄**：25至35岁之间
   - **特征**：双手指尖长有厚茧，手指微肌肉群饱满有力
   - **技能**：能熟练演奏钢琴一类乐器
   - **性格**：性情沉默寡淡，不喜欢抛头露面
   - **职业方向**：私人家教或钢琴调音师一类
   - **关键**：大概率不姓李

3. **寻找任务**
   - 皮阮通过关系网找人
   - 江城表示她们可能在任何一座城市
   - 找到之后的事与皮阮无关

4. **皮阮离开**
   - 皮阮问林老板是否来过，江城说没有
   - 皮阮离开时与胖子打招呼
   - 胖子拎菜篮子借口买菜溜了

5. **江城独处**
   - 胖子离开后，江城摆弄手机
   - 相册中有四张素描画的照片
   - 江城放大其中一张年轻女人的照片
   - 照片中女孩穿皮西米亚风裙子，十分单纯
   - 江城放大到只剩一双眼睛，隔着屏幕对视

6. **神秘女孩电话**
   - 电话那头龚叔焦急询问为什么没去考试
   - 女孩躺在床上，被子凌乱，说自己病了
   - 上周也是用这借口请假
   - 龚叔说下不为例
   - 女孩问龚叔叔让她办的事怎么样了

### 本章要点
- 江城展现出惊人的侧写分析能力
- 皮阮是江城的朋友网络资源
- 6亿资产三七开：皮阮拿1千8百万
- 江城对画中女人有特殊关注
- 出现新角色：龚叔和神秘女孩，暗示与江城要找的人有关
