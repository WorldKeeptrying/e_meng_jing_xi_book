# 第十一章-剧情

## 核心事件

### 二次探索与真相重构

别墅内幸存者基于鞋码和物品证据，推翻原有推测，重构案件真相。

### 关键发现

| 发现 | 位置 | 推论 |
|------|------|------|
| 女士休闲鞋（两双） | 一楼客厅鞋柜 | 别墅内有**两个女孩**（非一个） |
| 鞋底磨损差异 | 同一双鞋的左右脚 | 左鞋外侧磨损，右鞋内侧磨损 → 走路姿势不同 → **属于两个不同的人** |
| 拖鞋（尺码偏小） | 军事海报卧室床底 | 该卧室住的是女孩（非男孩） |
| 卫生巾 | 军事海报卧室抽屉 | 证实卧室住的是女性 |
| 破旧运动鞋（两双） | 鞋柜 | 一双接近江城尺码，一双比樊力还大 → **入侵者是两个成年男人** |

## 案情重构

### 原推测（错误）
- 四口之家（男女主人 + 两个孩子）
- 家庭暴力导致的惨剧

### 新推测（正确）
- **恶徒入侵案**
- 两个成年男人闯入别墅
- 杀死男女主人
- 囚禁并虐待两个女儿
- 最终杀死两个女孩

### 女鬼身份推测
- 根据浴缸中的女人头发 + 水渍痕迹
- 女鬼很可能是被**摁在浴缸中溺死**的女孩之一
- 谢雨被吓死后又被穿上入侵者的鞋拖走虐打 → 女鬼将谢雨**想象成了当初对她施虐的人**

## 江城的关键推理

### 别墅居住者构成
通过鞋子、拖鞋、卫生巾等证据，江城推理出：
- 主卧室：男女主人
- 二层小卧室1：一个女孩（性格偏向男性，房间贴军事海报）
- 二层小卧室2：另一个女孩
- **不存在男孩卧室**

### 入侵者身份
- 两双破旧运动鞋与别墅内其他鞋子格格不入
- 尺码显示是两个成年男人（非男孩）
- 鞋子廉价、破旧，与别墅主人的富裕程度不符

### 时间异常预警
- 江城察觉**白天越来越短暂**
- **黑夜变得愈发漫长**
- 估算：最多再经过**两个晚上**，白天就会**彻底消失**
- 胖子询问"白天消失后会发生什么？"
- 江城回答："大不了就是鬼会现身肆无忌惮的杀人"

## 角色表现

### 江城
- **推理能力极强**：通过鞋码、磨损、拖鞋、卫生巾等碎片线索重构案情
- **冷静分析**：面对恐怖的真相重构过程保持理性
- **预警意识**：注意到时间异常，提前警告同伴
- **幽默（黑色）**：安慰胖子"不用担心...大不了就是鬼会现身肆无忌惮的杀人"
- **谨慎**：拉胖子进卧室"因为我害怕"（实为单独搜查线索）

### 樊力
- **观察力敏锐**：通过鞋底磨损差异推断出两个女孩
- **领导气质**：继续组织搜索行动
- **沉稳**：面对真相重构不慌乱

### 暖姐
- **经验分享**：解释照片出现是"梦界"的正常现象（触发下一阶段剧情）
- **三层观察**：来时观察到别墅只有一二层有窗户，三层是暗房
- **常识判断**：卧室不会设在无窗暗房中

### 胖子
- **恐惧但跟随**：始终紧贴江城
- **信息提醒**：提到三层未探索

### 陈晓萌
- **推理辅助**：提出"两个男孩是外来者"的假设
- **情绪波动**：面对真相时身体哆嗦

## 新增线索

| 线索 | 意义 |
|------|------|
| 鞋子磨损差异 | 证明两个女孩（非一个） |
| 拖鞋 + 卫生巾 | 证明军事海报卧室住的是女孩 |
| 破旧运动鞋 | 证明入侵者是两个成年男人 |
| 时间异常 | 白天逐渐消失，黑夜变长 → 最终鬼可能肆无忌惮杀人 |
| 三层暗房 | 无人居住，但有探索价值 |
| 梦界规则（暖姐透露） | 找到正确线索会触发下一阶段剧情（类似游戏阶段奖励） |

## 世界观扩展

### 梦界规则
- 线索触发机制：找到正确线索 → 触发下一阶段剧情
- 类似游戏中的"阶段奖励"
- 照片会**突然出现**在之前空白的位置

### 时间机制
- 白天逐渐缩短
- 黑夜逐渐延长
- 预测：最多两个晚上后白天彻底消失
- 暗示：最终黑夜将完全统治，鬼的活动不受限制

## 悬念

- 白天彻底消失后会发生什么？
- 三层暗房内有什么？
- 两个入侵者的具体身份是什么？
- 两个女孩的详细遭遇是什么？
- 幸存者能否在白天消失前完成任务或逃离？
