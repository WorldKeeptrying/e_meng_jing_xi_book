# 第十八章 - 报纸

## 核心事件

### 1. 江城拆穿胖子意图
- 胖子想"和兄弟分在一起"只是借口
- **真实目的**：让江城带他过噩梦结界
- 逻辑：噩梦只有第一次与无数次之分
  - 这次有江城带，能活
  - 下次可能没这么好的运气

### 2. 江城工作室环境
- 办公桌放着几本病例本
- 胖子看到后问江城是不是医生
- 江城回答："主治各种不服"
- 沙发老旧，扶手干裂起皮
- 沙发一侧能放倒变成床（用于引导患者放松）

### 3. 胖子（王富贵）背景揭露

| 信息 | 内容 |
|------|------|
| **真名** | 王富贵 |
| **职业经历** | 什么工作都做过 |
| **最近工作** | 长途货车司机（已辞职） |
| **居住地** | 鹅城（距离榕城颇远） |
| **梦界经验** | 第一次进入噩梦世界 |
| **进入方式** | 公园墙上出现一扇不存在的门 |
| **进入前状态** | 被老板克扣工资，吵架后被赶出宿舍，没钱住旅馆，在公园准备对付到天亮 |
| **穿着** | 穿戴整齐，随身携带雨伞 |

### 4. 江城的态度
- 允许胖子待到天亮
- **天亮后必须离开**
- 态度坚决，无改变意思

### 5. 报纸内容（核心真相）

#### 报纸标题
**别墅虐杀案告破，凶徒一家坠江死亡**

#### 案件详情
| 项目 | 内容 |
|------|------|
| **罪犯一家** | 男女主人 + 两个女儿，都是异端信徒 |
| **作案方式** | 男主人开伪装成旅行大巴的客车骗受害者 |
| **分工** | 女儿1：伪装导游（郑导）<br>女儿2：网上发钓鱼帖<br>女主人：统筹大局 |
| **囚禁虐待** | 将受害者囚禁在别墅酷刑虐待 |
| **案件规模** | 短短一个月，警方接到多起报失案件 |
| **受害者** | 包括一对双胞胎兄弟 |
| **警方行动** | 汇总失踪者行动轨迹，顺藤摸瓜找到别墅 |
| **罪犯逃跑** | 先一步逃掉，带上3名受害者 |
| **坠江原因** | 连夜暴雨冲垮部分道路，大巴车速过快躲闪不及 |
| **死亡情况** | 罪犯一家四口 + 三名受害者全部溺水身亡 |
| **案发时间** | **2006年6月1日**（报纸日期）→ **14年前** |

#### 关键照片
- 溺亡人尸体（暂时无法确定身份）
- 死者特征：
  - 身材高大壮硕
  - 面部模糊（经过处理）
  - **黑色翻领皮衣十分显眼**
  - 随身物品：怀表
- **胖子认出怀表** → 江城打断："看样子像是樊力"

### 6. 梦界本质推断

#### 胖子的推测
- 每个梦界都是**真实存在过的**
- 那扇门是将人**送回过去**
- 重新体验那些人经历过的事情

#### 江城的修正
1. **孤例不证**：只经历一次噩梦，不能下结论
2. **更准确的说法**：
   - 即便门是送回过去
   - 重新体验的也仅仅是**受害者经历的惨案**
   - 或者说**另一场噩梦**
3. **噩梦结界的恐怖**：每一次都面临生死危机
4. **暖姐与樊力**：过了几次任务，仍惨死在厉鬼手中

### 7. 水渍之谜解答
- 胖子问：别墅中鬼出现的地方都有水渍？
- 江城答：因为作恶的一家人都是**溺水死的**
- 成鬼后保留这一特征

### 8. 报纸来源之谜
- 胖子问报纸从哪得来
- 江城撒谎："陈晓萌送我的"
- 胖子质疑："她烧了也不可能送你"
- **江城眼珠咕噜一转** → 明显在说谎

## 重要规则补充

### 梦界与过去的关系
1. 梦界可能是真实存在的历史事件
2. 门可能是送回过去的通道
3. 体验的是受害者的惨案/噩梦
4. 在梦界死亡可能死于过去时空（樊力）

### 鬼的特征
- 溺水而死的鬼会出现水渍
- 鬼保留死亡时的特征

### 物品携带
- 从梦界可以带回物品（报纸）
- 报纸在现实中有对应历史

## 伏笔与悬念

1. **报纸真实来源** - 江城为什么撒谎？报纸从哪来？
2. **樊力确认死亡？** - 只是"像是"，未完全确认
3. **双胞胎兄弟** - 受害者中的双胞胎，后续是否出现？
4. **胖子的去留** - 天亮后真会离开？还是会留下？
5. **下一次梦界** - 胖子需要人带，江城会同意吗？
6. **孤例不证** - 还会有更多梦界任务验证理论？
